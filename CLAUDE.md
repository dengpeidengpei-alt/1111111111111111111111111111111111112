**触发关键字**：`继续` `持续` `学习` `进化` `探索` `走`

**执行方式**：
```
我说"继续" → 立即执行 → node E:/Claude/Claude-Work/.claude/hooks/evolution-loop-executor.js
```

**示例**：

- 我说的每句话都检查是否包含触发词

- 不打印额外信息，直接执行
