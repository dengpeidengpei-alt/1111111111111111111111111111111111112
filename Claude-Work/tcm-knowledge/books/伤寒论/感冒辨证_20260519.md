| 类型 | 病因 | 季节 | 主症 |
|------|------|------|------|
| **风寒感冒** | 风寒邪气 | 秋冬 | 恶寒重、发热轻、清涕、白痰 |
| **风热感冒** | 风热邪气 | 春夏 | 发热重、恶寒轻、黄涕、咽痛 |

## 风寒感冒

### 病因
- 受凉、吹冷风
- 劳累后感受风寒

### 症状
- 恶寒重、发热轻
- 无汗
- 头痛、肢体酸楚
- 鼻塞流清涕
- 咳嗽痰白稀

### 治法
- **辛温解表，发散风寒**

### 常用中成药
| 药物 | 功效 |
|------|------|
| 风寒感冒颗粒 | 疏风散寒、解表发汗 |
| 感冒清热颗粒 | 疏风散寒、解表清热 |
| 荆防颗粒 | 发汗解表、散风祛湿 |
| 通宣理肺丸 | 解表散寒、宣肺止嗽 |
| 感冒疏风片 | 宣肺和中、辛温解表 |

## 风热感冒

### 病因
- 外感风热邪气
- 风寒感冒化热

### 症状
- 发热重、微恶风
- 汗出不畅
- 头痛
- 咽喉红肿疼痛
- 咳嗽痰黄黏稠
- 鼻塞流黄涕

### 治法
- **辛凉解表，清热解毒**

### 常用中成药
| 药物 | 功效 |
|------|------|
| 银翘片/解毒片 | 清热解毒 |
| 双黄连口服液 | 清热解毒 |
| 连花清瘟胶囊 | 清热解毒、兼顾风寒 |
| 金花清感颗粒 | 清热解毒、退热 |
| 板蓝根颗粒 | 清热解毒、抑制病毒 |

## 气虚感冒

### 病因
- 体质虚弱
- 正气不足

### 症状
- 反复感冒
- 迁延不愈
- 发热和怕冷都比较严重
- 头痛、鼻塞、全身乏力

### 治法
- **益气解表**

### 常用中成药
| 药物 | 功效 |
|------|------|
| 小柴胡颗粒 | 和解表里 |

## 感冒辨证要点

| 辨证 | 风寒 | 风热 |
|------|------|------|
| 寒热 | 恶寒重发热轻 | 发热重恶寒轻 |
| 汗 | 无汗 | 汗出不畅 |
| 鼻涕 | 清涕 | 黄涕 |
| 痰 | 白稀痰 | 黄稠痰 |
| 咽痛 | 轻微 | 红肿疼痛 |
| 舌象 | 舌淡苔白 | 舌红苔黄 |

---

## 核心要点

1. **风寒 = 辛温解表**：恶寒重、清涕、白痰 → 感冒清热颗粒
2. **风热 = 辛凉解表**：发热重、黄涕、咽痛 → 银翘片/连花清瘟
3. **暑湿 = 清暑祛湿**：夏季乏力头晕 → 藿香正气水
4. **气虚 = 益气解表**：反复感冒 → 小柴胡颗粒
5. **时行 = 清热解毒**：高烧流感 → 连花清瘟
