## 肝气郁结概述

**病因**：情志不遂、肝气郁结
**病机**：肝主疏泄，气机不畅
**治法**：疏肝解郁

---

## 症状表现

| 部位 | 症状 |
|------|------|
| 胁肋 | 胁肋胀痛、胸闷 |
| 情绪 | 抑郁寡欢、烦躁易怒 |
| 消化 | 食欲不振、消化不良 |
| 全身 | 叹息、乏力 |
| 妇科 | 月经不调、乳房胀痛 |
| 舌脉 | 舌淡红、脉弦 |

## 常用方剂

### 逍遥散

**组成**：
| 药物 | 功效 |
|------|------|
| 柴胡 | 疏肝解郁（君） |
| 当归 | 养血活血 |
| 白芍 | 养血柔肝 |
| 白术 | 健脾益气 |
| 茯苓 | 健脾渗湿 |
| 甘草 | 益气补中 |
| 生姜 | 温胃和中 |
| 薄荷 | 疏散郁热 |

**功效**：疏肝解郁、养血健脾

**主治**：
- 肝郁血虚脾弱证
- 两胁作痛
- 头痛目眩
- 口燥咽干
- 神疲食少
- 月经不调
- 乳房胀痛

**现代应用**：
- 慢性肝炎
- 肝硬化
- 胆石症
- 胃及十二指肠溃疡
- 经前期紧张症
- 乳腺小叶增生

---

### 柴胡疏肝散

**组成**：陈皮、柴胡、川芎、香附、枳壳、芍药、甘草

**功效**：疏肝解郁、理气畅中

**主治**：肝郁气滞引起的胁肋疼痛、精神抑郁、嗳气、腹胀

---

### 四逆散

**功效**：透邪解郁、疏肝理脾

**主治**：手足畏冷、忽寒忽热

---

### 丹栀逍遥散

**组成**：逍遥散 + 丹皮、栀子

**功效**：清肝泻火

**主治**：肝郁化热、急躁易怒
