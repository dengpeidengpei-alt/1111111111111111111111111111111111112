## 春季养肝

### 春季特点
- 万物复苏，阳气升发
- 肝气旺盛，易肝郁/肝火旺

### 养生要点
| 方面 | 方法 |
|------|------|
| **饮食** | 绿色蔬菜（菠菜、芹菜、荠菜）、疏肝理气（枸杞菊花茶） |
| **运动** | 太极拳、散步、踏青 |
| **起居** | 早睡早起，避免熬夜 |
| **情志** | 保持心情舒畅，避免抑郁愤怒 |

### 常见问题
- 肝气郁结：情绪低落、胁胀失眠 → 柴胡疏肝散
- 肝火旺盛：头晕头痛、烦躁易怒 → 龙胆泻肝汤

---

## 夏季养心

### 夏季特点
- 烈日炎炎，阳气外越
- 心火亢盛，易心烦失眠

### 养生要点
| 方面 | 方法 |
|------|------|
| **饮食** | 苦味食物（苦瓜、莲子心）、绿豆、冬瓜、清热解暑 |
| **运动** | 游泳、瑜伽、傍晚散步 |
| **起居** | 晚睡早起，午间小憩，防暑降温 |
| **情志** | 心情平静，忌怒 |

### 常见问题
- 暑湿之邪：口干口渴、心烦意乱 → 藿香正气水、参苓白术散
- 心火亢盛：失眠烦躁 → 天王补心丹

## 秋季养肺

### 秋季特点
- 气候干燥，燥邪犯肺
- 肺气易损，易干咳

### 养生要点
| 方面 | 方法 |
|------|------|
| **饮食** | 白色食物（梨、百合、银耳）、润肺养阴 |
| **运动** | 慢跑、深呼吸练习 |
| **起居** | 保持室内湿度，多喝水，早晚保暖 |
| **情志** | 心情舒畅，避免悲伤 |

### 常见问题
- 肺燥热：咳嗽、咽干、口干 → 沙参麦冬汤、养肺丸

---

## 冬季养肾

### 冬季特点
- 寒气凝滞，阳气内藏
- 肾阳易损，需封藏

### 养生要点
| 方面 | 方法 |
|------|------|
| **饮食** | 温补食物（黑豆、核桃、羊肉）、补肾固精 |
| **运动** | 室内运动为主，避免大汗 |
| **起居** | 早睡晚起，必待日光保暖 |
| **情志** | 藏精养神，避免恐惧 |

### 常见问题
- 肾阳虚：腰膝酸软、畏寒怕冷 → 右归丸
- 肾阴虚：手足心热、盗汗 → 左归丸

## 四季养脾胃

### 脾胃重要性
- 脾胃是将饮食转化为气血的关键
- 气血是人体活动的基本物质

### 养生要点
| 季节 | 方法 |
|------|------|
| 四季 | 健脾和胃 |
| 饮食 | 定时定量，少食生冷 |
| 运动 | 适量运动，促进消化 |

### 常见问题
- 气血不足：疲倦乏力、自汗 → 当归补气丸
