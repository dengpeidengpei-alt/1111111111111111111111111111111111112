## 治未病概述

**出处**：《黄帝内经》
**核心思想**：未病先防，既病防变

---

## 治未病三层含义

| 层次 | 说明 | 方法 |
|------|------|------|
| **未病先防** | 疾病发生前预防 | 增强正气，避免外邪 |
| **既病防变** | 疾病发生后防止传变 | 及时治疗，阻断加重 |
| **瘥后防复** | 病愈后防止复发 | 继续调养，巩固疗效 |

---

## 未病先防

### 原则
1. **法于自然之道**：顺应四时变化
2. **调理精神情志**：保持心情安宁
3. **保持阴平阳秘**：调和阴阳

### 具体方法
| 方面 | 方法 |
|------|------|
| **饮食** | 五谷为养、五果为助、五畜为益、五菜为充 |
| **起居** | 规律作息，春夏晚睡早起，秋冬早睡晚起 |
| **运动** | 太极拳、八段锦，适量运动 |
| **情志** | 心情舒畅，避免过度的精神压力 |
| **环境卫生** | 讲卫生，避免污染 |

### 四季起居
| 季节 | 起居要求 |
|------|----------|
| 春 | 晚睡早起，庭院散步，舒缓身体 |
| 夏 | 晚睡早起，不要厌恶白天太长 |
| 秋 | 早睡早起，保持意志安定 |
| 冬 | 早睡晚起，避开寒凉保温 |

---

## 既病防变

### 原则
- 疾病初期及时干预
- 预测疾病发展方向
- 阻断疾病加重或传变

### 示例
| 疾病阶段 | 处理方法 |
|----------|----------|
| 感冒初期 | 葱豉汤发散表邪，避免入里化热 |
| 糖尿病 | 配合饮食运动管理，预防并发症 |

---

## 瘥后防复

### 原则
- 大病初愈继续调理
- 防止疾病复发

### 方法
| 方法 | 说明 |
|------|------|
| 健脾益气 | 四君子汤恢复正气 |
| 饮食调养 | 补充营养、适当休息 |
| 定期评估 | 健康评估，警惕复发 |

---

## 核心要点

1. **治未病 = 未病先防 + 既病防变 + 瘥后防复**
2. **未病先防 = 增强正气 + 规避外邪**
3. **既病防变 = 及时治疗 + 阻断传变**
4. **瘥后防复 = 继续调养 + 防止复发**
5. **正气存内，邪不可干**
