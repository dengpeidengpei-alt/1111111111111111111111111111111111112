## 煎药器具

| 推荐 | 禁用 |
|------|------|
| 砂锅 | 铁锅 |
| 搪瓷器皿 | 铜锅 |
| 不锈钢 | 铝锅 |

**原因**：金属器具可能与药物发生反应，影响药效甚至产生有害物质

---

## 煎药前准备

### 浸泡
- **时间**：30分钟至2小时
- **水量**：浸过药材表面2-3厘米
- **注意**：用冷水，不用热水

### 煎煮次数
- 一般煎煮2-3次
- 两次药液混合后分次服用

## 特殊药材处理

### 先煎
- **药物**：矿物类、贝壳类（磁石、牡蛎、石膏）
- **目的**：延长煎煮时间，利于有效成分溶出
- **时间**：20-30分钟，或遵医嘱

### 后下
- **药物**：含挥发性成分（薄荷、荆芥、豆蔻、砂仁）
- **目的**：减少挥发性成分损失
- **时间**：煎煮快结束时加入（最后5-10分钟）

### 包煎
- **药物**：粉末状、绒毛类（蒲黄、海金沙）
- **目的**：避免药液混浊
- **方法**：用纱布包裹后煎煮

### 烊化
- **药物**：胶类（阿胶、鹿角胶）
- **目的**：单独融化后兑入药液

### 冲服
- **药物**：贵重细粉（三七粉、川贝粉）
- **方法**：用药液冲服

## 服药方法

- **温度**：温热服最好
- **频率**：每日1剂，分2次（早晚各1）
- **儿童**：小口分次服
- **服用间隔**：与西药隔开1-2小时

## 服药忌口

| 忌口食物 | 原因 |
|----------|------|
| 生冷 | 影响药效 |
| 辛辣 | 加重热象 |
| 油腻 | 碍胃 |
| 腥膻 | 影响药性 |
| 浓茶 | 成分反应 |
| 牛奶 | 间隔1-2小时 |

## 核心要点

1. **器具 = 砂锅/陶瓷，忌铁铝**
2. **浸泡 = 30分钟至2小时，冷水**
3. **火候 = 先武火煮沸，后文火慢煎**
4. **时间 = 解表15-20分，补益30-40分**
5. **特殊处理 = 先煎/后下/包煎/烊化/冲服**
6. **服药 = 温热服，忌生冷辛辣油腻**
