## 外因：六淫

### 六淫概述
- **六气**：风寒暑湿燥火（正常气候）
- **六淫**：六气太过/不及，成为致病因素
- **特点**：多与季节气候、生活环境有关

### 风邪
| 特点 | 说明 |
|------|------|
| 风为阳邪 | 其性开泄，易袭阳位 |
| 善行数变 | 病位游走不定（荨麻疹、关节游走痛） |
| 风性主动 | 动摇不定 |
| 风为百病之长 | 易夹杂其他邪气 |

### 寒邪
| 特点 | 说明 |
|------|------|
| 寒为阴邪 | 易伤阳气 |
| 寒性凝滞 | 导致疼痛 |
| 寒性收引 | 痉挛、怕冷 |

### 暑邪
| 特点 | 说明 |
|------|------|
| 暑为阳邪 | 其性炎热 |
| 暑性升散 | 扰神伤津耗气 |
| 暑多挟湿 | 常见暑湿夹攻 |

### 湿邪
| 特点 | 说明 |
|------|------|
| 湿为阴邪 | 损伤阳气，阻遏气机 |
| 湿性重浊 | 头身困重、便粘 |
| 湿性黏滞 | 病程缠绵 |
| 湿性趋下 | 易袭阴位 |

### 燥邪
| 特点 | 说明 |
|------|------|
| 燥性干涩 | 易伤津液 |
| 燥易伤肺 | 口干鼻干、干咳 |

### 火邪（热邪）
| 特点 | 说明 |
|------|------|
| 火为阳邪 | 其性趋上 |
| 火热扰心神 | 心烦失眠 |
| 火热伤津耗气 | 口渴、乏力 |
| 火热易生风动血 | 高热抽搐、出血 |
| 易致疮痈 | 红肿热痛 |

## 内因：七情

### 七情概述
- **七情**：喜怒忧思悲恐惊
- **正常**：情志活动不会致病
- **异常**：强烈持久的情志刺激 → 七情内伤

### 七情伤脏
| 情志 | 所伤脏腑 |
|------|----------|
| 怒伤肝 | 怒则气上 |
| 喜伤心 | 喜则气缓 |
| 思伤脾 | 思则气结 |
| 忧伤肺 | 悲忧则气郁气消 |
| 恐伤肾 | 恐则气下 |
| 惊伤肾 | 惊则气乱 |
| 悲伤心 | 同忧 |

### 致病特点
1. **直接伤及内脏**：怒伤肝、喜伤心、思伤脾、忧伤肺、恐伤肾
2. **影响脏腑气机**：气上、气缓、气郁、气消、气下、气乱、气结
3. **数情交织**：多伤及心肝脾
4. **加重病情**：情绪波动可致病情恶化

## 不内外因

| 类型 | 后果 |
|------|------|
| 过饥 | 气血亏虚、抗病力下降 |
| 过饱 | 脾胃受损、消化不良、肥胖 |
| 饮食不洁 | 脏腑功能失调 |

## 核心要点

### 六淫
1. **风邪 = 善行数变 + 百病之长**
2. **寒邪 = 凝滞 + 收引**
3. **暑邪 = 炎热 + 升散 + 挟湿**
4. **湿邪 = 重浊 + 黏滞 + 趋下**
5. **燥邪 = 干涩 + 易伤肺**
6. **火邪 = 炎上 + 耗气伤津 + 生风动血**

### 七情
1. **怒伤肝 → 气上**
2. **喜伤心 → 气缓**
3. **思伤脾 → 气结**
4. **忧伤肺 → 气消**
5. **恐伤肾 → 气下**
6. **惊伤肾 → 气乱**
