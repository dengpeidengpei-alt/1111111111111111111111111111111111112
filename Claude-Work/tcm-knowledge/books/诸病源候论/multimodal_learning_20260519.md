## 知识入库

### 1. 基本信息

**书名**：诸病源候论（又称《巢氏病源》）
**作者**：巢元方（隋）
**成书**：610年（大业六年）
**规模**：
- 50卷
- 67门
- 1720论（1739论）
- 中国现存第一部病因病机、证候学专著

**地位**：
- 官纂医书，太医博士奉诏编撰
- 历代医家重视，唐《外台秘要》、宋《太平圣惠方》多引用

### 2. 病因学创新

**突破前人学说**：

| 疾病 | 前人学说 | 巢元方创新 |
|------|----------|------------|
| 传染病 | 六淫之说 | "乖戾之气"，可互相传染 |
| 疥疮 | 不明 | 确认疥虫病原（针头挑得） |
| 寄生虫病 | 不详 | 饮食不洁而生 |
| 炭疽 | 不明 | 先有疮而接触病畜传染 |
| 漆过敏 | 不明 | 与人禀性素质有关 |

**环境病因认识**：
- 山区"瘿"病 → "饮沙水"所致
- 岭南"瘴气" → "杂毒因暖而生"

### 3. 病机理论

**核心理论**：正气虚损
- 正气不足 → 卫外不固
- 六淫、疠气等外邪乘虚而入 → 引发诸病

### 4. 证候学成就

**记载精细**：
- 消渴病并发痈疖或水肿（糖尿病并发症最早描述）
- 脚气病描述："自膝至脚有不仁、或苦痹、或淫淫如虫所缘、或脚趾及膝胫洒洒尔、或脚屈弱不能行、或微肿、或酷冷而疼痛"
- 水肿分述至详

### 5. 学术贡献

**分科论述**：
| 科 | 卷 | 规模 |
|----|-----|------|
| 内科 | 卷1~27 | 论述最详 |
| 五官科 | 卷28~30 | - |
| 外伤科 | 卷31~36 | - |
| 妇产科 | 卷37~44 | - |
| 小儿科 | 卷45~50 | - |

**治疗特点**：
- 每证末多附导引法
- 但不记载治疗方药（专论病源、候）
