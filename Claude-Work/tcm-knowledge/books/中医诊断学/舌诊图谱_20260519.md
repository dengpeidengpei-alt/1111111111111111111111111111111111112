## 正常舌象

**标准**：淡红舌、薄白苔
- 舌色淡红鲜明，舌质滋润
- 舌体大小适中，不胖不瘦，柔软灵活
- 舌苔薄白均匀，干湿适中，不滑不燥

---

## 三种典型舌象

### 1. 肾虚舌象
| 特征 | 描述 |
|------|------|
| 舌苔 | 舌根部厚腻，黄白色，湿热或寒湿蕴结 |
| 舌体 | 颜色稍暗，舌质苍老 |
| 舌象 | 舌根反映肾脏，苔厚=肾气不足 |

**关联症状**：腰膝酸软、男科功能下降

### 2. 脾虚舌象
| 特征 | 描述 |
|------|------|
| 舌体 | 胖大肿胀，舌边齿痕（受牙齿挤压） |
| 舌色 | 淡红或偏白，缺乏红润 |
| 舌苔 | 薄白且润，湿润 |

**关联症状**：食欲不振、腹胀、消化不良

### 3. 肝郁舌象
| 特征 | 描述 |
|------|------|
| 舌体 | 紧致尖细，不胖大 |
| 舌色 | 两侧（肝胆区）明显较红，有红点 |
| 裂纹 | 舌面中心纵向裂纹（中缝） |
| 舌苔 | 薄白不均，中部略黄燥 |

**关联症状**：失眠多梦、脾气急躁易怒

---

## 舌诊要点

### 望舌质（舌体）
1. **舌色**：淡白→淡红→红→绛→紫
2. **舌形**：老嫩、胖瘦、芒刺、裂纹、齿痕
3. **舌态**：痿软、强硬、颤动、歪斜、短缩

### 望舌苔
1. **苔色**：白→黄→灰黑
2. **苔质**：薄厚、润燥、腐腻

### 舌象与证候

| 舌象 | 主证 |
|------|------|
| 淡白舌 | 气血两虚、阳虚 |
| 红舌 | 热证 |
| 绛舌 | 热盛伤津 |
| 紫舌 | 瘀血、寒证 |
| 齿痕舌 | 脾虚湿盛 |
| 裂纹舌 | 精血亏虚 |
| 胖大舌 | 脾虚、肾虚水泛 |

---

## 核心要点

1. **淡红舌薄白苔 = 正常**
2. **舌根候肾，舌边候肝胆，舌中候脾胃**
3. **齿痕舌 = 脾虚湿盛**
4. **裂纹舌 = 精血亏虚**
5. **舌色越深，热邪越盛**
