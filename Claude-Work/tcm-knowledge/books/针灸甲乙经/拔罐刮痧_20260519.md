# 拔罐疗法

## 拔罐原理

**定义**：借助热力排除罐中空气，利用负压使其吸着于皮肤，造成瘀血现象的一种方法。

**原理**：
- 物理刺激和负压造成毛细血管破裂瘀血
- 调动人体修复功能
- 促进血液循环，激发精气
- 调整人体阴阳平衡

## 拔罐功效

| 功效 | 说明 |
|------|------|
| **通经活络** | 疏通经络 |
| **行气活血** | 促进气血运行 |
| **消肿止痛** | 消除肿胀 |
| **祛风散寒** | 驱散风寒湿邪 |
| **拔毒泻热** | 清热解毒 |
| **祛除淤滞** | 消除瘀滞 |
| **解除疲劳** | 增强体质 |
| **扶正祛邪** | 提高免疫力 |

## 常用拔罐部位

| 部位 | 适应症 |
|------|--------|
| 肩背部 | 肩背痛、腰痛 |
| 腰骶部 | 痛经、腰腿痛 |
| 腹部 | 腹痛、胃痛 |
| 阿是穴 | 各种疼痛 |

## 拔罐方法

| 方法 | 操作 |
|------|------|
| 火罐 | 用火排除空气（闪火罐最常用） |
| 抽气罐 | 用抽气工具抽出空气 |
| 水罐 | 水中煮沸后吸拔 |

## 拔罐时间

- **一般**：10分钟左右
- **儿童/老人**：需缩短时间
- **禁忌**：不超过20分钟，否则可能起泡

# 刮痧疗法

## 刮痧原理

**定义**：用刮痧板在皮肤相关部位刮拭，以达到活血化瘀、排毒排浊的治疗方法。

**原理**：
- 疏通经络
- 活血化瘀
- 排出毒素
- 促进代谢

## 刮痧功效

| 功效 | 说明 |
|------|------|
| **活血化瘀** | 改善局部微循环 |
| **清热解毒** | 清除热毒 |
| **舒筋通络** | 缓解肌肉紧张 |
| **调整阴阳** | 平衡阴阳 |
| **排毒美容** | 改善皮肤状态 |

## 常用刮痧部位

| 部位 | 适应症 |
|------|--------|
| 颈部 | 感冒、头痛、颈椎病 |
| 背部 | 背痛、感冒、发热 |
| 上肢 | 肩周炎、手臂麻木 |
| 下肢 | 腿痛、坐骨神经痛 |
| 面部 | 美容、面部疾病 |

## 刮痧手法

| 手法 | 操作 |
|------|------|
| 角刮法 | 用刮痧板一角刮拭 |
| 面刮法 | 用刮痧板平面刮拭 |
| 点按法 | 用角部点按穴位 |

# 注意事项

## 拔罐注意
1. **保暖**：拔罐时注意避风寒
2. **防烫伤**：火罐操作时注意烫伤
3. **时间控制**：一般10分钟，不超过20分钟
4. **起泡处理**：消毒后无菌处理
5. **晕罐处理**：立即取罐平卧

## 刮痧注意
1. **介质**：使用刮痧油或清水
2. **力度**：均匀柔和，避免用力过猛
3. **方向**：顺着经络走向
4. **程度**：出痧即可，不可强行出痧
