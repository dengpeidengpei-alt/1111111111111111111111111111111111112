## 艾灸概述

**定义**：用艾叶制成的艾灸材料产生艾热，刺激体表穴位或特定部位，通过激发经气的活动来调整紊乱的人体功能，达到防病治病目的。

**历史**：产生于中国远古石器时代，文献最早见于《庄子》

---

## 艾灸功效

| 功效 | 说明 |
|------|------|
| **温经散寒** | 治疗寒凝经络引发的腹痛、痛经、寒疝 |
| **活血化瘀** | 促进气血运行，治疗寒湿血滞 |
| **扶阳固脱** | 升阳举陷，治疗脱肛、久泻、崩漏 |
| **消瘀散结** | 用于外科疮疡初起 |
| **拔毒泻热** | 清热泻火（需辨证） |
| **调节阴阳** | 平衡机体阴阳 |
| **防病保健** | 增强免疫 |

## 艾灸方法

### 直接灸
- 艾炷直接放在皮肤上施灸
- 可分为化脓灸和非化脓灸

### 间接灸
- 隔姜灸、隔蒜灸、隔盐灸等
- 减少直接烧伤

### 艾条灸
- 温和灸、回旋灸、雀啄灸
- 适合家庭保健

## 艾灸禁忌

### 禁用部位/情况
- **面部、颈部、大血管**：不可直接灸
- **乳头、会阴部**：禁止施灸
- **妊娠期腰骶部、下腹部**：绝对禁忌
- **皮肤感染、溃疡**：禁灸
- **局部有炎症**：避免施灸

### 禁用人群/状态
| 状态 | 原因 |
|------|------|
| **阴虚内热** | 伤阴动火 |
| **实热证** | 迫血妄行 |
| **外感温热病** | 加重热象 |
| **醉酒** | 气血逆乱 |
| **过饥过饱** | 脾胃失调 |
| **过度劳累** | 正气耗伤 |
| **情绪不稳** | 气机紊乱 |
| **出血倾向** | 加重出血 |
| **传染病/皮肤病** | 加重病情 |
| **昏迷/感觉迟钝** | 剂量不可过大 |

### 特殊人群
- **糖尿病患者**：需在医师指导下
- **凝血功能障碍者**：需在医师指导下
- **精神病患者**：不可自灸

## 注意事项

1. **灸后保暖**：避免吹风受寒
2. **避免立即洗澡**：至少等待1-2小时
3. **不过度重灸**：时间不宜过长
4. **辨证施灸**：实热体质者缩短时间、减少频率
5. **专业指导**：应由专业医师操作
