# Knowledge Graph & LLM Reasoning

**学习时间**: 2026-05-20 20:35
**来源**: 互联网搜索

---

## Knowledge Graph 核心概念

**定义**: 结构化的知识表示形式，通过实体和关系构建语义丰富的知识网络

---

## Think-on-Graph (ToG) 范式

**核心**: LLM ⊗ KG 紧耦合范式

**原理**:
- LLM作为智能代理与知识图谱协同工作
- 通过波束搜索动态探索推理路径
- 迭代执行：初始化→探索→剪枝

---

## KG在LLM中的作用

| 作用 | 说明 |
|------|------|
| **补充缺失信息** | KG中的三元组补充LLM固有知识 |
| **结构化推理** | 基于图的结构化知识 |
| **可解释性** | 推理路径可追溯 |
| **过滤验证** | 作为过滤器和验证器 |
