## Common Cross-ref Issues

Many cross-refs fail because the target key doesn't match exactly. Common patterns:

1. **Chinese names with spaces**: `八正散` vs `八 正散`
2. **Similar but different**: `当归补血汤` vs `当归`
3. **Nested TCM items**: 药材(如`人参`) vs 方剂(如`四君子汤`)

### Fix Approach

For TCM entries, cross-refs should use:
- 药材直接引用：如 `[[tcm/herbs/人参.m[[knowledge/Design-Toolkit]]]` → `[[tcm/herbs/人参.m[[knowledge/Design-Toolkit]]] — 人参`
- 方剂引用：如 `[[四君子汤]]` → `[[四君子汤]] — 四君子汤`
- 不要跨类别引用：如 药材不要引用方剂
