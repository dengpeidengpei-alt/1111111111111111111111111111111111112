# 诸病源候论

## Core Stats
- **1720条** 病因证候论述
- **巢元方**/隋代（605-616年）
- 中医病因证候学奠基之作
- 《巢氏病源》简称

## Key Concepts

### 病因学 (Etiology)
- **外感六淫** — 风寒暑湿燥火各有所伤
- **内伤七情** — 喜伤心/怒伤肝/思伤脾/忧伤肺/恐伤肾
- **饮食劳倦** — 醉饱入房/强力举重
- **虫兽金刃** — 意外伤害

### 病机学 (Pathogenesis)
- **邪正相争** — 决定发病与否
- **阴阳失衡** — 疾病性质总纲
- **脏腑失调** — 病位传变的内在机制

### 证候学 (Syndrome Classification)
- **证候群描述** — 症状组合规律
- **病候演变** — 证候转化与传变
- **预后判断** — 证候轻重与生死

## 学术贡献

1. **病因学系统化** — 首次分门别类论述病因
2. **证候学规范化** — 症状描述标准化
3. **病机学理论化** — 建立病因-病机-证候对应关系
4. **养生预防体系** — 重视调摄防病

## 学术地位

- **中医病因证候学鼻祖**
- 继《黄帝内经》后病因学最大突破
- 宋代《太平圣惠方》多从此出
- 明清温病学的重要源头
