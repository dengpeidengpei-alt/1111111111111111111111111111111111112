# 金匮要略

## Overview
- **经典地位**: 中医杂病辨证之祖，张仲景《伤寒杂病论》残卷
- **核心贡献**: 病脉证治体系、杂病辨证范式、方剂配伍规范
- **结构**: 25篇，262方，3卷
- **与伤寒论关系**: 同源著作，伤寒论论外感，金匮论内伤杂病

## 杂病辨治

### 百合狐惑阴阳毒

**百合病**
```
主证：神志恍惚、饮食行为异常
病机：心肺阴虚燥热
治法：滋阴润燥
主方：百合地黄汤
加减：
- 误吐后 → 百合鸡子汤
- 误下后 → 百合代赭汤
- 误汗后 → 百合知母汤
```

**狐惑病**
```
主证：口腔咽喉溃疡、目赤、皮肤损害
病机：湿热虫毒
治法：清热燥湿
主方：甘草泻心汤
外治：苦参汤熏洗
```

**阴阳毒**
```
分类：
- 阳毒 → 面部红斑、咽喉痛、脉数 → 升麻鳖甲汤
- 阴毒 → 面色青、咽喉痛、脉沉 → 升麻鳖甲去雄黄蜀椒汤
```

### 疟病

| 证型 | 主证 | 治法 | 方剂 |
|------|------|------|------|
| 牝疟 | 寒多热少 | 温阳散寒 | 柴胡桂姜汤 |
| 温疟 | 热多寒少 | 清热截疟 | 白虎加桂枝汤 |
| 瘴疟 | 但热不寒 | 益气生津 | 蜀漆散 |

### 中风历节

**中风**
```
主证：半身不遂、口眼歪斜、语言不利
病机：风痰阻络
治法：祛风化痰
代表方：
- 风引汤 — 除热瘫痫
- 侯氏黑散 — 补虚祛风
- 续命汤 — 温通经络
```

**历节**
```
主证：关节肿痛、变形
病机：肝肾不足、风湿侵袭
治法：祛风除湿
代表方：
- 桂枝芍药知母汤 — 寒热错杂
- 乌头汤 — 寒盛痛剧
```

### 血痹虚劳

**血痹**
```
主证：肢体麻木、肌肤不仁
病机：气血不足、外感风邪
治法：益气和营
方剂：黄芪桂枝五物汤
```

**虚劳**
```
分类：
- 气虚 → 四君子汤系列
- 血虚 → 四物汤系列
- 阴阳两虚 → 桂枝加龙骨牡蛎汤
- 虚劳腰痛 → 八味肾气丸
- 虚劳诸不足 → 薯蓣丸

代表方：小建中汤、黄芪建中汤
```

### 肺痿肺痈咳嗽上气

**肺痿**
```
分类：
- 虚寒肺痿 → 甘草干姜汤
- 虚热肺痿 → 麦门冬汤
```

**肺痈**
```
阶段：
1. 表期 → 脉浮、咳嗽 → 麻黄杏仁甘草石膏汤
2. 成脓期 → 脉数、吐脓 → 桔梗汤
3. 溃后期 → 吐脓血 → 《千金》苇茎汤
```

**咳嗽上气**
```
代表方：射干麻黄汤、越婢加半夏汤、小青龙加石膏汤
```

### 胸痹心痛短气

```
代表方：栝楼薤白白酒汤
病机：胸阳不振、痰浊闭阻

栝楼薤白白酒汤 → 胸痹轻证
栝楼薤白半夏汤 → 胸痹痰盛
枳实薤白桂枝汤 → 胸痹气结
人参汤 → 胸痹中气虚
茯苓杏仁甘草汤 → 胸痹短气
橘枳姜汤 → 胸痹气滞
```

### 腹满寒疝宿食

**腹满**
```
辨证：
- 实热 → 厚朴三物汤、大承气汤
- 寒实 → 大黄附子汤、温脾汤
- 寒热错杂 → 附子粳米汤
```

**寒疝**
```
主证：腹部冷痛、拘急
代表方：
- 大乌头煎 — 寒疝核心方
- 当归生姜羊肉汤 — 血虚寒疝
- 乌头桂枝汤 — 表里俱寒
```

| 溢饮 | 四肢 | 发汗解表 | 大青龙汤、小青龙汤 |

### 水气病

| 分类 | 特点 | 治法 | 代表方 |
|------|------|------|--------|
| 风水 | 眼睑浮肿 | 祛风利水 | 越婢汤 |
| 皮水 | 四肢浮肿 | 利水消肿 | 防己茯苓汤 |
| 正水 | 腹大 | 温阳利水 | 麻黄附子汤 |
| 石水 | 脐下硬 | 温下逐水 | 桂枝去芍药加麻黄细辛附子汤 |

### 黄疸病

```
分类：
- 谷疸 → 食谷即眩 → 茵陈蒿汤
- 酒疸 → 心中懊憹 → 栀子大黄汤
- 女劳疸 → 额黑微汗 → 硝石矾石散
- 黑疸 → 久病转归

治法：清热利湿、退黄
主方：茵陈蒿汤、栀子大黄汤
兼证：血瘀 → 抵挡汤
```

### 惊悸吐衄下血瘀血

**惊悸**
```
治法：温阳化饮
- 心阳虚 → 桂枝去芍药加蜀漆龙骨牡蛎救逆汤
- 心血虚 → 酸枣仁汤
```

**吐衄下血**
```
治法：
- 虚寒便血 → 黄土汤
- 热盛衄血 → 泻心汤
- 瘀血 → 柏叶汤
```

### 呕吐哕下利

**呕吐**
```
代表方：
- 小半夏汤 — 胃寒呕吐
- 大半夏汤 — 虚寒呕吐
- 吴茱萸汤 — 肝寒犯胃
- 黄芩加半夏生姜汤 — 胆热犯胃
- 半夏泻心汤 — 寒热错杂
```

**下利**
```
代表方：
- 虚寒 → 四逆汤、理中汤
- 实热 → 承气汤
- 寒热错杂 → 乌梅丸
```

### 疮痈肠痈浸淫病

**肠痈**
```
代表方：大黄牡丹皮汤
主治：右下腹疼痛、发热
脓成 → 薏苡附子败酱散
```

**浸淫疮**
```
代表方：黄连粉
主治：皮肤溃烂、渗液
```

### 跌蹶手指臂肿转筋阴狐疝蛔虫

**蛔虫**
```
代表方：乌梅丸
主治：腹痛时作、吐蛔
安蛔法：寒温并用、酸苦辛甘
```

## 经典方剂详解

### 八味肾气丸（崔氏八味丸）

```
组成：干地黄8两 山茱萸4两 薯蓣4两 泽泻3两
      茯苓3两 牡丹皮3两 桂枝1两 附子1两

功效：温补肾阳
主治：肾阳不足、腰痛脚软、小便不利

病机分析：
┌─────────────────────────────────────────────────────────────────────┐
│                        肾气丸方解                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   附子+桂枝 ──→ 温补肾阳 ──→ 君药                                   │
│       │                                                             │
│       ├──→ 干地黄 ──→ 滋阴补肾 ──→ 臣药                             │
│       │              └──→ 阴中求阳                                  │
│       │                                                             │
│       ├──→ 山茱萸+薯蓣 ──→ 肝肾同补 ──→ 佐药                        │
│       │                                                             │
│       └──→ 泽泻+茯苓+丹皮 ──→ 渗湿泻浊 ──→ 使药                      │
│                               └──→ 补而不滞                         │
│                                                                     │
│   配伍特点：阴中求阳、少火生气                                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 桂枝茯苓丸

```
组成：桂枝 茯苓 丹皮 桃仁 芍药（各等分）

功效：活血化瘀、缓消癥块
主治：妇人癥病、胞衣不下

病机：瘀血内停
方解：
- 桂枝 — 温通血脉
- 桃仁 — 活血破瘀
- 丹皮 — 凉血散瘀
- 芍药 — 和营缓急
- 茯苓 — 渗湿健脾

应用：子宫肌瘤、卵巢囊肿、痛经
```

### 当归芍药散

```
组成：当归三两 芍药一斤 茯苓四两 白术四两
      泽泻半斤 川芎半斤

功效：养血调肝、健脾利湿
主治：妇人怀妊腹中㽱痛

病机：肝血不足、脾虚湿盛
方解：
- 当归+川芎+芍药 — 养血柔肝
- 茯苓+白术+泽泻 — 健脾利湿
```

### 百合地黄汤

```
组成：百合七枚 生地黄汁一升

功效：滋阴润燥
主治：百合病（神志恍惚）

病机：心肺阴虚燥热
服法：泉水浸渍、煎服
反应：服后大便如漆（地黄颜色）
```

## 方剂配伍原则

### 君臣佐使规范

| 角色 | 定义 | 举例 |
|------|------|------|
| 君药 | 主病之药 | 桂枝汤中桂枝 |
| 臣药 | 辅君之药 | 桂枝汤中芍药 |
| 佐药 | 佐君之偏 | 防止药偏 |
| 使药 | 引经之药 | 调和诸药 |

### 煎服方法

| 方法 | 代表方 | 要求 |
|------|--------|------|
| 先煎 | 麻黄、葛根 | 减轻毒性 |
| 后下 | 薄荷、大黄 | 保全药力 |
| 烊化 | 阿胶、饴糖 | 另炖或烊化 |
| 去滓再煎 | 小柴胡汤 | 协调诸药 |
