# 黄帝内经

## Overview
- **地位**: 中医理论奠基之作
- **成书**: 战国至东汉（公元前475年—公元220年）
- **字数**: 约20万字
- **结构**: 《素问》81篇 + 《灵枢》81篇 = 162篇
- **价值**: 中医理论的源头，被称为"医之始祖"

### 2. 阴阳学说

```
阴阳关系：

对立制约 ←→ 互根互用
    ↓              ↓
 寒者热之    ←→   阴在内阳之守
 热者寒之         阳在外阴之使

消长平衡 ←→ 相互转化
    ↓              ↓
 阳胜则热     ←→   重阴必阳
 阴胜则寒         重阳必阴
```

## 治病原则

### 治未病
```
未病先防 → 养生保健
    ↓
既病防变 → 早期治疗
    ↓
瘥后防复 → 病后调理
```

### 治病求本
- **正治**：寒者热之，热者寒之
- **反治**：热因热用，寒因寒用

### 三因制宜
- 因时制宜 — 根据季节
- 因地制宜 — 根据地域
- 因人制宜 — 根据体质

## 养生智慧

### 四季养生
```
春（生）→ 夜卧早起，广步于庭，被发缓形
夏（长）→ 夜卧早起，无厌于日
秋（收）→ 早卧早起，与鸡俱兴
冬（藏）→ 早卧晚起，必待日光
```
