# 伤寒论

## Overview
- **经典地位**: 中医临床之祖，张仲景《伤寒杂病论》残卷
- **核心贡献**: 六经辨证体系、方证对应原则、辨病论治范式
- **结构**: 398条条文，112方，10卷
- **学派**: 伤寒学派，温病学派（延伸发展）

## 临床应用

### 太阳病辨治

#### 桂枝汤证（太阳中风）
```
主证：发热、汗出、恶风、脉浮缓
病机：风寒袭表、营卫不和
治法：解肌发表、调和营卫
方药：桂枝9g 芍药9g 生姜9g 大枣12枚 甘草6g
服法：微火煮取，温服，啜热粥，覆被取汗
禁忌：禁生冷、粘滑、肉面等（共计五忌）
```

#### 麻黄汤证（太阳伤寒）
```
主证：恶寒发热、无汗而喘、脉浮紧
病机：风寒束表、肺气不宣
治法：发汗解表、宣肺平喘
方药：麻黄9g 桂枝6g 杏仁9g 甘草3g
禁忌：疮、淋、血、尺脉微者禁之
```

### 少阳病辨治

#### 小柴胡汤证
```
主证：寒热往来、胸胁苦满、默默不欲饮食、心烦喜呕
病机：邪犯少阳、枢机不利
治法：和解少阳
方药：柴胡24g 黄芩9g 人参9g 半夏9g 甘草9g
      生姜9g 大枣12枚
加减：
- 胸中烦而不呕 → 去半夏、人参，加瓜蒌
- 渴 → 去半夏，加天花粉
- 胁下硬 → 去大枣，加牡蛎
```

### 阳明病辨治

#### 白虎汤证
```
主证：大热、大汗、大渴、脉洪大
病机：阳明气分热盛
治法：清热生津
方药：石膏50g 知母18g 甘草6g 粳米9g
煎服法：先煮石膏，余药后下，煮至米熟汤成
禁忌：脉沉迟、不可与；脉浮弦、不可与
```

#### 承气汤证
```
分类：
1. 调胃承气汤 — 燥结在胃，微和胃气
   大黄12g 芒硝12g 甘草6g

2. 小承气汤 — 燥结在肠，轻下热结
   大黄12g 厚朴6g 枳实9g

3. 大承气汤 — 燥结在直肠，峻下热结
   大黄12g 厚朴15g 枳实12g 芒硝9g

适应症：
- 潮热
- 谵语
- 手足濈然汗出
- 燥屎内结
```

## 条文精选

### 辨太阳病脉证并治（上）

**第1条**：太阳病，提纲
```
太阳病，脉浮，头项强痛而恶寒。
```

**第2条**：太阳中风
```
太阳病，发热，汗出，恶风，脉缓者，名为中风。
```

**第3条**：太阳伤寒
```
太阳病，或已发热，或未发热，必恶寒，体痛，呕逆，脉阴阳俱紧者，名为伤寒。
```

**第12条**：桂枝汤证
```
太阳中风，阳浮而阴弱，阳浮者热自发，阴弱者汗自出，
啬啬恶寒，淅淅恶风，翕翕发热，鼻鸣干呕者，桂枝汤主之。
```

**第13条**：桂枝汤证
```
太阳病，头痛，发热，汗出，恶风，桂枝汤主之。
```

**第14条**：桂枝加葛根汤证
```
太阳病，项背强几几，反汗出恶风者，桂枝加葛根汤主之。
```

**第16条**：桂枝汤禁忌
```
桂枝本为解肌，若其人脉浮紧，发热汗不出者，不可与之也。
常须识此，勿令误也。
```

**第25条**：桂枝汤服法
```
服桂枝汤，大汗出，脉洪大者，与桂枝汤，如前法。
```

**第28条**：去桂加茯苓白术汤
```
服桂枝汤，或下之，仍头项强痛，翕翕发热，无汗，心下满微痛，
小便不利者，桂枝去桂加茯苓白术汤主之。
```

### 辨少阳病脉证并治

**第263条**：少阳病提纲
```
少阳之为病，口苦，咽干，目眩也。
```

**第264条**：少阳禁吐下
```
少阳中风，两耳无所闻，目赤，胸中满而烦者，不可吐下，吐下则悸而惊。
```

**第265条**：少阳禁汗
```
伤寒，脉弦细，头痛发热者，属少阳。
少阳不可发汗，发汗则谵语，此属胃，胃和则愈，胃不和则烦而悸。
```

**第266条**：小柴胡汤证
```
本太阳病不解，转入少阳者，胁下硬满，干呕不能食，往来寒热，
尚未吐下，脉沉紧者，与小柴胡汤。
```

**第267条**：柴胡汤疑似证
```
若已吐、下、发汗、温针，谵语，柴胡汤证罢，此为坏病，
知犯何逆，以法治之。
```

### 辨阳明病脉证并治

**第180条**：阳明病提纲
```
阳明病，胃家实是也。
```

**第182条**：阳明外证
```
问曰：阳明病外证云何？答曰：身热，汗自出，不恶寒，反恶热也。
```

**第185条**：阳明初病
```
病有得之一日，不发热而恶寒者，何也？答曰：虽得之一日，恶寒将自罢，即自汗出而恶热也。
```

**第219条**：白虎汤证
```
三阳合病，腹满，身重，难以转侧，口不仁，面垢，谵语，遗尿，
发汗则谵语，下之则额上生汗，手足逆冷，若自汗出者，白虎汤主之。
```

**第220条**：承气汤证
```
二阳并病，太阳证罢，但发潮热，手足漐漐汗出，大便难而谵语者，下之则愈，宜大承气汤。
```

**第221条**：栀子豉汤证
```
阳明病，脉浮而紧，咽燥，口苦，腹满而喘，发热汗出，不恶寒，反恶热，身重，
若发汗则躁，心愦愦，反谵语；若加温针，必怵惕，烦躁不得眠；
若下之，则胃中空虚，客气动膈，心中懊憹，舌上苔者，栀子豉汤主之。
```

## 经典医案

### 医案一：桂枝汤治疗太阳中风

```
患者：张某，男，35岁
主诉：发热汗出3天
病史：3天前受凉后出现发热，体温38.5℃，汗出恶风，
      头痛，项强，食欲不振，二便正常。
查体：舌淡红，苔薄白，脉浮缓
诊断：太阳中风（桂枝汤证）
处方：桂枝9g 白芍9g 生姜9g 大枣12枚 甘草6g
      啜粥覆被取汗
复诊：1剂后汗出热退，3剂后诸症消失
按：风寒袭表，营卫不和，桂枝汤解肌发表、调和营卫而愈
```

### 医案二：小柴胡汤治疗少阳病

```
患者：李某，女，42岁
主诉：寒热往来半月
病史：半月来每日午后寒热往来，体温37.5-38.2℃，
      伴胸胁胀满，默默不欲食，心烦喜呕，口苦咽干。
查体：舌红，苔薄黄，脉弦数
诊断：少阳病（小柴胡汤证）
处方：柴胡24g 黄芩9g 人参9g 半夏9g
      甘草9g 生姜9g 大枣12枚
复诊：3剂后寒热止，7剂后诸症消失
按：邪犯少阳，枢机不利，小柴胡汤和解少阳而愈
```

### 医案三：白虎汤治疗阳明经热

```
患者：王某，男，28岁
主诉：高热大渴3天
病史：3天前酒后外感，高热39.8℃，大汗，大渴喜冷饮，
      面红目赤，烦躁不安，小便短赤，大便干结。
查体：舌红，苔黄燥，脉洪大数
诊断：阳明气分热盛（白虎汤证）
处方：石膏50g 知母18g 甘草6g 粳米9g
复诊：1剂后热退渴减，3剂后痊愈
按：阳明气分热盛，白虎汤清热生津而愈
```
