# 千金方

## Core Stats
- **4054首方剂** — 中医方剂之冠
- **孙思邈**/唐代（581-682年）
- 第一部《千金要方》卷帙浩繁
- 第二部《千金翼方》补充深化

## Key Principles

### 补虚益精
- **养生延年** — "人命至重，有贵千金"
- 虚劳七伤 → 补益肝肾
- 精血亏损 → 滋阴填髓

### 大医精诚
- 医德规范奠基之作
- "大慈恻隐之心，普救含灵之苦"
- 反对炫技邀荣、诋毁同行

### 综合性方剂全书
- 妇儿五官内外科全覆盖
- 针灸按摩导引并重
- 食疗养生预防一体
