# 复脉汤

## Origin
- **温病条辨** — 吴鞠通
- **分类** — 补益剂·滋阴剂
- **地位** — 下焦病补阴主方，脉结代救命方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 炙甘草 | 12g | 君药 | 益气复脉，补中缓急 |
| 干地黄 | 18g | 臣药 | 滋阴补血，益精填髓 |
| 生白芍 | 18g | 臣药 | 养血敛阴，柔肝舒筋 |
| 麦冬 | 10g | 佐药 | 滋阴润燥，清心除烦 |
| 阿胶 | 9g | 佐药 | 补血止血，滋阴润燥 |
| 麻仁 | 10g | 佐药 | 润肠通便，滋阴养血 |

## 功效
- **滋阴养血** — 补益下焦阴液
- **复脉救逆** — 恢复脉道，救逆扶危

## 方解配伍图

```
炙甘草（君）→ 益气复脉，补中缓急（复脉要药）
     ↓
干地黄+生白芍（臣）→ 滋阴补血，柔肝舒筋（补益肝肾）
     ↓
麦冬+阿胶（佐）→ 滋阴润燥，清心除烦（心肺胃阴）
     ↓
麻仁（佐）→ 润肠通便，滋阴养血（润燥滑肠）
     ↓
配伍特点：滋阴为主，益气为辅；复脉救逆，标本兼顾
```

## 主治

### 经典症状（温病条辨）
> 手下焦温病，热邪久羁，脉结代，舌绛苔少，时欲脱者，复脉汤主之。

**典型症状**：
- **脉结代** — 脉律不齐，时有中止
- **舌绛** — 舌质红绛，阴伤之象
- **口干** — 阴液耗伤
- **心悸** — 心阴不足
- **时有欲脱** — 气阴两脱征兆

### 病机分析
```
热邪久羁 → 阴液耗伤 → 脉道不荣
     ↓
心阴不足 → 心悸不安，脉结代
     ↓
治法：滋阴养血，益气复脉
```

## 条文精选

### 温病条辨·下焦篇
> 温病脉法，不似伤寒论之促。结代脉者，以脉之阴阳两虚，邪正相争也。当以下焦温病热邪久羁，脉结代为主治。

### 伤寒论第182条（炙甘草汤类似）
> 伤寒，脉结代，心动悸，炙甘草汤主之。

## 临床应用（现代医学）

| 疾病 | 机理 | 应用 |
|------|------|------|
| 心律不齐 | 益气复脉 | 室性早搏、房颤 |
| 病毒性心肌炎 | 滋阴养血 | 心阴不足型 |
| 扩张型心肌病 | 补益气阴 | 气阴两虚型 |
| 心力衰竭 | 复脉救逆 | 心阴耗伤型 |
| 更年期综合征 | 滋阴养血 | 心悸失眠型 |

## 医案

### 医案1：心律不齐
```
患者：女性，52岁
主诉：心悸半年，脉结代
症状：心悸不安，失眠多梦，舌红少苔
心电图：频发室性早搏
辨证：心阴不足，气血两虚
治法：滋阴养血，益气复脉
方药：复脉汤加苦参、丹参
效果：2周后早搏减少，继续调理3月心电图正常
```

### 医案2：病毒性心肌炎
```
患者：男性，28岁
主诉：感冒后心悸2周
症状：心悸气短，失眠多梦，舌红少津
脉象：结代脉
辨证：热病后期，心阴耗伤
治法：滋阴养血，益气复脉
方药：复脉汤加太子参、五味子
效果：4周后症状消失，心电图恢复正常
```

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 复脉汤 | 原方 | 下焦阴伤脉结代 |
| 加减复脉汤 | 去麻仁，加牡蛎 | 阴虚风动 |
| 炙甘草汤 | 复脉汤+人参桂枝生姜 | 气血两虚心悸 |
| 生脉散 | 人参麦冬五味子 | 气阴两虚 |
| 温胆汤 | 复脉汤+二陈汤 | 痰热扰心 |

## 使用注意

| 情况 | 处理 |
|------|------|
| 阳虚寒盛 | 不宜使用（需温阳） |
| 痰浊内阻 | 先化痰再滋阴 |
| 脾虚便溏 | 麻仁润肠，慎用 |
| 外感发热 | 不宜使用 |
