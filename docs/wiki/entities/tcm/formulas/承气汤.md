# 承气汤

## Origin
- **伤寒论** — 张仲景
- **分类** — 泻下剂·寒下剂
- **地位** — 阳明腑实代表方系列，峻下热结经典

## 组成（分类与剂量）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|

## 功效
- **峻下热结** — 大承气汤
- **轻下热结** — 小承气汤
- **缓下热结** — 调胃承气汤

## 方解配伍图

### 大承气汤组方结构
```
大黄（君）→ 泻下热结，荡涤肠胃（推陈致新）
     ↕（相须为用）
芒硝（臣）→ 软坚润燥，助大黄泻下（润燥软坚）
     ↓
枳实+厚朴（佐）→ 行气破结，消痞除满（宽肠下气）
     ↓
配伍特点：硝黄相须，枳朴相使；峻下热结，急下存阴
```

### 三承气汤鉴别
```
痞满燥实 → 大承气汤（燥实明显）
     ↓（轻剂）
痞满实 → 小承气汤（实而不燥）
     ↓（缓剂）
燥实 → 调胃承气汤（胃气不和）
```

## 主治

### 阳明腑实四大证（伤寒论第182条）
> 问曰：阳明病外证云何？答曰：身热，汗自出，不恶寒，反恶热也。

```
痞 — 脘腹痞满（脾胃气滞）
满 — 腹满胀痛（燥屎内结）
燥 — 肠中燥屎（津液耗伤）
实 — 便秘腹痛（腑气不通）
```

### 病机分析
```
阳明热盛 → 腑气不通 → 燥屎内结
     ↓
热结肠道 → 腹痛便秘，神昏谵语
     ↓
治法：峻下热结，急下存阴
```

## 条文精选

### 伤寒论第182条
> 问曰：阳明病外证云何？答曰：身热，汗自出，不恶寒，反恶热也。

### 伤寒论第209条
> 阳明病，潮热，大便微硬者，可与大承气汤；不硬者，不可与之。若不大便六七日，恐有燥屎，欲知之法，少与小承气汤，汤入腹中，转失气者，此有燥屎也，乃可攻之。

### 伤寒论第215条
> 阳明病，谵语，有潮热，反不能食者，胃中必有燥屎五六枚也。若能食者，但硬耳。宜大承气汤下之。

## 医案

### 医案1：急性肠梗阻
```
患者：男性，58岁
主诉：腹痛腹胀3天，停止排气排便
症状：腹部胀满疼痛，烦躁谵语，发热
舌象：舌红苔黄燥
脉象：脉弦数有力
辨证：阳明腑实，燥屎内结
治法：峻下热结，行气导滞
方药：大承气汤原方
效果：1剂后排出燥屎，腹痛减轻，继续调理痊愈
```

### 医案2：急性胰腺炎
```
患者：女性，42岁
主诉：上腹痛2天，伴呕吐，发热
症状：腹痛拒按，大便不通，尿黄
舌象：舌红苔黄腻
脉象：脉弦数
辨证：阳明腑实，湿热内蕴
治法：泻下热结，清热利湿
方药：大承气汤加柴胡、黄芩、栀子
效果：2剂后大便通，腹痛减，热退，继续清热调理
```

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 大承气汤 | 硝黄枳朴 | 峻下热结（燥实明显） |
| 小承气汤 | 去芒硝 | 轻下热结（痞满为主） |
| 调胃承气汤 | 去枳朴，加甘草 | 缓下热结（胃气不和） |
| 桃核承气汤 | 大黄+桃仁桂枝 | 瘀血热结 |
| 增液承气汤 | 大黄+玄参麦冬生地 | 热结阴伤 |
