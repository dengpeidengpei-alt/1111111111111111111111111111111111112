# 八珍汤

## Origin
- **正体类要**（明代·薛己）
- **分类** — 补益剂·气血双补
- **地位** — 气血双补代表方，四君子+四物复合方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 人参 | 9g | 君药 | 大补元气，健脾益肺 |
| 白术 | 9g | 臣药 | 健脾益气，燥湿利水 |
| 茯苓 | 9g | 佐药 | 健脾安神，利水渗湿 |
| 甘草 | 6g | 使药 | 益气和中，调和诸药 |
| 熟地黄 | 12g | 君药 | 补血滋阴，益精填髓 |
| 当归 | 10g | 臣药 | 补血活血，调经止痛 |
| 白芍 | 12g | 佐药 | 养血敛阴，柔肝止痛 |
| 川芎 | 8g | 使药 | 活血行气，祛风止痛 |

## 功效
- **益气补血** — 气血双补，治本之法
- **健脾养心** — 健脾助运，养心安神

```

    ↓

    ↓

```

## 主治

### 经典病机
```
气血两虚 → 脏腑失养 → 功能衰退
     ↓
心脾两虚 → 心悸失眠，食欲不振
     ↓
治法：益气补血，健脾养心
```

**典型症状**：
- **面色苍白或萎黄** — 气血亏虚，失于荣养
- **头晕眼花** — 血虚不养清窍
- **心悸失眠** — 心脾两虚，气血不足
- **食欲不振** — 脾气亏虚，运化无力
- **舌淡苔白，脉细弱** — 气血两虚之象

## 临床应用（现代医学）

| 疾病 | 机理 | 应用 |
|------|------|------|
| 贫血 | 益气补血 | 缺铁性贫血、气血两虚 |
| 手术后恢复 | 气血双补 | 术后体虚调理 |
| 产后调理 | 补益气血 | 产后失血恢复 |
| 月经不调 | 调血养肝 | 气血两虚型 |
| 神经衰弱 | 健脾养心 | 心脾两虚型失眠 |

## 医案

### 医案1：贫血
```
患者：女性，35岁
主诉：乏力半年，血红蛋白90g/L
症状：面色萎黄，头晕眼花，心悸失眠
舌象：舌淡苔白
脉象：脉细弱
辨证：气血两虚，心脾两亏
治法：益气补血，健脾养心
方药：八珍汤加阿胶、黄芪
效果：2月后血红蛋白升至120g/L，症状消失
```

### 医案2：产后调理
```
患者：女性，28岁
主诉：产后3月，乏力、乳汁不足
症状：面色苍白，心悸多汗，食欲不振
舌象：舌淡苔白
脉象：脉细弱
辨证：气血两虚，产后失养
治法：益气补血，健脾生乳
方药：八珍汤加黄芪、王不留行
效果：1月后乳汁充足，体力恢复
```

## 使用注意

| 情况 | 处理 |
|------|------|
| 阴虚火旺 | 不宜使用（补益助火） |

| 外感发热 | 暂停服用 |
| 糖尿病 | 减少熟地黄用量 |
