# 小柴胡汤

## Origin
- **伤寒论** — 张仲景
- **分类** — 和解剂代表方
- **地位** — 伤寒少阳病主方，中医十大名方之一

## 组成（剂量比例）

| 药材 | 剂量 | 角色 |
|------|------|------|
| 柴胡 | 24g | 君药 — 疏散少阳邪气 |
| 黄芩 | 9g | 臣药 — 清少阳郁热 |
| 人参 | 9g | 佐药 — 益气扶正 |
| 半夏 | 9g | 佐药 — 和胃降逆 |
| 生姜 | 9g | 佐药 — 和胃止呕 |
| 大枣 | 6枚 | 使药 — 调和脾胃 |
| 甘草 | 6g | 使药 — 调和诸药 |

## 功效
- **和解少阳** — 核心功效
- **疏肝和胃** — 次级功效
- **扶正祛邪** — 兼顾正气

## 主治

### 经典症状（伤寒论）
- **寒热往来** — 忽冷忽热，往来交替
- **胸胁苦满** — 胸胁部胀满不适
- **心烦喜呕** — 心情烦躁，恶心欲吐
- **嘿嘿不欲饮食** — 食欲不振，情绪低落

### 病机分析
```
病位：半表半里（少阳胆经）
病性：邪正相争
治法：和解少阳
```

## 六经辨证定位

```
太阳病 → 表证 → 麻黄汤/桂枝汤
    ↓
阳明病 → 里热实证 → 白虎汤/承气汤
    ↓
少阳病 → 半表半里 → 小柴胡汤 ← 我们在这里
    ↓
太阴病 → 里虚寒证 → 理中汤
    ↓
少阴病 → 心肾阳虚 → 四逆汤
    ↓
厥阴病 → 寒热错杂 → 乌梅丸
```

## 现代应用

| 疾病 | 应用 | 机理 |
|------|------|------|
| 感冒中期 | 寒热往来型 | 和解少阳 |
| 慢性肝炎 | 肝郁气滞 | 疏肝和胃 |
| 胆囊炎 | 肝胆湿热 | 清肝利胆 |
| 胃炎 | 肝胃不和 | 和胃降逆 |
| 更年期综合征 | 寒热往来 | 调和阴阳 |

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 小柴胡汤 | 原方 | 少阳病基础方 |
| 大柴胡汤 | 去人参，加芍药、大黄 | 少阳兼阳明腑实 |
| 柴胡桂枝汤 | 小柴胡+桂枝汤 | 太阳少阳合病 |
| 柴胡加龙骨牡蛎汤 | 加龙骨、牡蛎、铅丹 | 少阳痰热扰心 |
| 柴胡疏肝散 | 去黄芩、人参，加香附、枳壳 | 肝气郁结 |

## 临床案例

### 案例1：感冒中期
```
症状：忽冷忽热3天，伴胸胁胀满、恶心
辨证：少阳病
治法：和解少阳
方药：小柴胡汤原方
效果：1剂症状减半，3剂痊愈
```

### 案例2：慢性胃炎
```
症状：胃脘胀痛，情绪相关，食欲不振
辨证：肝胃不和
治法：疏肝和胃
方药：小柴胡汤加减（加香附、枳壳）
```

## 使用注意

| 情况 | 处理 |
|------|------|
| 无少阳证据 | 不宜使用 |
| 阴虚火旺 | 慎用或加减 |
| 孕妇 | 慎用（柴胡升散） |
| 食积腹痛 | 先消积后和解 |
