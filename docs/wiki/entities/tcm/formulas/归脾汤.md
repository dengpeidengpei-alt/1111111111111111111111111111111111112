# 归脾汤

## Origin
- **济生方**（宋代·严用和）
- **分类** — 补益剂·补血剂
- **地位** — 心脾两虚代表方，健脾养心经典方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 人参 | 9g | 君药 | 大补元气，健脾益心 |
| 黄芪 | 12g | 臣药 | 补气升阳，益卫固表 |
| 白术 | 9g | 臣药 | 健脾益气，燥湿利水 |
| 茯神 | 9g | 佐药 | 健脾安神，宁心安神 |
| 酸枣仁 | 12g | 佐药 | 养血安神，补肝宁心 |
| 龙眼肉 | 12g | 佐药 | 补益心脾，养血安神 |
| 木香 | 6g | 佐药 | 行气止痛，健脾和胃 |
| 当归 | 9g | 佐药 | 补血活血，养心安神 |
| 远志 | 6g | 使药 | 安神益智，交通心肾 |
| 炙甘草 | 6g | 使药 | 益气补中，调和诸药 |

## 功效
- **益气补血** — 补益心脾气血
- **健脾养心** — 健脾运化，养心安神


## 主治

### 经典病机
```
心脾两虚 → 心血不足，脾气亏虚
     ↓
心失所养 → 心悸失眠，健忘多梦
脾失健运 → 食欲不振，乏力倦怠
     ↓
治法：益气补血，健脾养心
```

**典型症状**：
- **心悸失眠** — 心血不足，心神不宁
- **健忘多梦** — 心脾两虚，神不守舍
- **食欲不振** — 脾气亏虚，运化无力
- **乏力倦怠** — 气虚失养
- **舌淡苔白，脉细弱** — 气血两虚之象

### 与八珍汤鉴别
```
归脾汤：心脾两虚（心悸失眠为主）
八珍汤：气血两虚（面色萎黄为主）
```

## 临床应用（现代医学）

| 疾病 | 机理 | 应用 |
|------|------|------|
| 神经衰弱 | 健脾养心 | 心脾两虚型失眠 |
| 心律失常 | 补益心脾 | 心脾两虚型早搏 |
| 胃炎 | 健脾和胃 | 脾气虚型胃脘痛 |
| 功能性消化不良 | 健脾助运 | 脾虚气滞型 |
| 妇女更年期综合征 | 补益心脾 | 心脾两虚型 |

## 医案

### 医案1：神经衰弱失眠
```
患者：女性，42岁
主诉：失眠多梦3年
症状：难以入睡，心悸健忘，食欲不振，倦怠乏力
舌象：舌淡苔白
脉象：脉细弱
辨证：心脾两虚，气血不足
治法：健脾养心，益气补血
方药：归脾汤加夜交藤、合欢皮
效果：2周后睡眠改善，1月后心悸消失，继续调理2月痊愈
```

### 医案2：功能性消化不良
```
患者：男性，35岁
主诉：胃脘胀满半年，伴失眠
症状：餐后胀满，失眠多梦，食欲不振
舌象：舌淡苔白腻
脉象：脉细弱
辨证：心脾两虚，气滞湿阻
治法：健脾和胃，养心安神
方药：归脾汤去龙眼肉，加陈皮、砂仁
效果：1月后胃脘胀满减轻，睡眠改善
```
