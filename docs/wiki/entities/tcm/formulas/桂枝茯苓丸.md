# 桂枝茯苓丸

## Origin
- **金匮要略** — 张仲景
- **分类** — 祛瘀剂·活血化瘀
- **地位** — 缓消癥块代表方，妇科癥瘕常用方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 桂枝 | 9g | 君药 | 温通经脉，活血化瘀 |
| 茯苓 | 9g | 臣药 | 利水渗湿，健脾安神 |
| 牡丹皮 | 9g | 臣药 | 清热凉血，活血化瘀 |
| 桃仁 | 9g | 佐药 | 破血祛瘀，润肠通便 |
| 白芍 | 9g | 佐药 | 养血敛阴，缓急止痛 |

## 功效
- **活血化瘀** — 温通经脉，化解瘀血
- **缓消癥块** — 渐消缓散，不伤正气
- **寒温并用** — 桂枝温通，牡丹皮凉血

## 方解配伍图

```
桂枝（君）→ 温通经脉，活血化瘀（温通活血）
     ↓
牡丹皮（臣）→ 清热凉血，活血化瘀（凉血止血）
     ↕（寒温并济）
桃仁（佐）→ 破血祛瘀，润肠通便（逐瘀润下）
     ↓
茯苓（臣）→ 利水渗湿，健脾安神（利湿健脾）
     ↓
白芍（佐）→ 养血敛阴，缓急止痛（养血柔肝）
     ↓
配伍特点：温通与凉血并用，祛瘀与利湿同行；缓消癥块，不伤正气
```

## 主治

### 经典症状（金匮要略）
> 妇人宿有癥病，经断未及三月而得漏下不止，胎动在脐上者，为癥痼害。妊娠六月动者，前三月经水利时，胎也。下血者，后断三月杯也。所以血不止者，其癥不去故也。当下其癥，桂枝茯苓丸主之。

**典型症状**：
- **癥瘕积聚** — 腹部肿块，固定不移
- **漏下不止** — 阴道不规则出血
- **胎动不安** — 妊娠胎动异常
- **舌暗或有瘀斑** — 血瘀舌象
- **脉涩** — 瘀血阻滞

### 病机分析
```
瘀血内停 → 癥瘕形成 → 气血失调
     ↓
经脉不通 → 漏下不止，胎动异常
     ↓
治法：活血化瘀，缓消癥块
```

## 医案

### 医案1：子宫肌瘤
```
患者：女性，38岁
主诉：体检发现子宫肌瘤3cm，伴月经量多
症状：经期延长10天，色暗有块，小腹胀痛
舌象：舌暗红苔白
脉象：脉涩
辨证：瘀血内停，癥瘕形成
治法：活血化瘀，缓消癥块
方药：桂枝茯苓丸原方
效果：3个月后复查肌瘤缩小至2cm，月经量减少
```

### 医案2：卵巢囊肿
```
患者：女性，32岁
主诉：B超发现右侧卵巢囊肿4cm
症状：右小腹胀痛，经期加重，月经后期
舌象：舌暗苔白
脉象：脉弦涩
辨证：瘀血阻滞，冲任失调
治法：活血化瘀，散结消肿
方药：桂枝茯苓丸加红花、三棱、莪术
效果：2月后复查囊肿消失
```

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 桂枝茯苓丸 | 原方 | 瘀血癥瘕基础方 |
| 桂枝茯苓丸加大黄 | 加大黄 | 瘀血内结较重 |
| 桂枝茯苓丸加三棱莪术 | 加活血破气药 | 癥块较大 |
| 血府逐瘀汤 | 较桂枝茯苓丸更峻 | 胸中血瘀 |
| 膈下逐瘀汤 | 腹部瘀血 | 腹中瘀血 |

## 使用注意

| 情况 | 处理 |
|------|------|
| 孕妇出血 | 需辨证，瘀血所致可用 |
| 出血量多 | 需配止血药 |
| 脾胃虚寒 | 配健脾和胃药 |
| 癥块过大 | 需加破血药或手术 |
