# 大定风珠

## Origin
- **温病条辨** — 吴鞠通
- **分类** — 治风剂·滋阴熄风
- **地位** — 真阴耗竭动风代表方，危重证救命方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 生白芍 | 18g | 君药 | 养血敛阴，柔肝舒筋 |
| 阿胶 | 9g | 臣药 | 补血止血，滋阴润燥 |
| 龟板 | 12g | 臣药 | 滋阴潜阳，固本熄风 |
| 干地黄 | 18g | 臣药 | 滋阴补肾，益精填髓 |
| 麻仁 | 6g | 佐药 | 润燥通便，滋阴养血 |
| 五味子 | 6g | 佐药 | 敛肺滋肾，生津止渴 |
| 麦冬 | 18g | 佐药 | 滋阴润燥，清心除烦 |
| 鳖甲 | 12g | 佐药 | 滋阴潜阳，软坚散结 |
| 牡蛎 | 12g | 佐药 | 重镇潜阳，熄风止痉 |
| 甘草 | 12g | 使药 | 调和诸药，益气缓急 |
| 鸡子黄 | 2枚 | 特殊 | 滋阴润燥，养血熄风 |

## 功效
- **滋阴熄风** — 核心功效，填补真阴
- **重镇潜阳** — 介类潜阳，镇摄虚风

## 方解配伍图

```
生白芍+阿胶（君）→ 养血敛阴，柔肝熄风（补肝血）
     ↓
龟板+鳖甲（臣）→ 滋阴潜阳，固本熄风（补肾阴）
干地黄+麦冬（臣）→ 滋阴补肾，清心除烦（补心阴）
     ↓
牡蛎+鸡子黄（佐）→ 重镇潜阳，滋阴润燥（熄虚风）
     ↓
五味子+麻仁（佐）→ 敛阴生津，润燥通便（防重镇伤正）
     ↓
甘草（使）→ 调和诸药，益气缓急
     ↓
配伍特点：血肉有情，介类潜阳；滋阴熄风，重镇潜阳
```

## 主治

### 经典症状（温病条辨）
> 温病，真阴大亏，虚风内动，手指蠕动，神倦瘛疭，舌干绛，脉虚弱者，大定风珠主之。

**典型症状**：
- **手指蠕动** — 肝风内动，手指微微抽动
- **神倦瘛疭** — 精神倦怠，抽搐无力
- **舌干绛** — 舌质红绛，津液枯竭
- **脉虚弱** — 脉来虚弱，真阴耗竭
- **时时欲脱** — 元气虚脱征兆

### 病机分析
```
热病日久 → 真阴耗竭 → 肝失所养
     ↓
虚风内动 → 手指蠕动，瘛疭抽搐
     ↓
治法：滋阴填精，重镇熄风
```

## 条文精选

> 热邪深入下焦，脉结代者，以复脉汤主之；舌干绛者，以大定风珠主之。

## 医案

### 医案1：帕金森病
```
患者：男性，65岁
主诉：双手震颤3年，加重2月
症状：双手静止性震颤，手指蠕动，精神倦怠
舌象：舌红绛少苔
脉象：脉虚弱
辨证：真阴耗竭，虚风内动
治法：滋阴熄风，重镇潜阳
方药：大定风珠原方
效果：2月后震颤减轻，继续调理1年症状稳定
```

### 医案2：病毒性脑炎后遗症
```
患者：儿童，10岁
主诉：高热后四肢抽搐2月
症状：手指蠕动，神倦瘛疭，舌干绛
舌象：舌红绛无苔
脉象：脉虚弱
辨证：热病伤阴，虚风内动
治法：滋阴熄风，益气复脉
方药：大定风珠加太子参、黄芪
效果：1月后抽搐减少，3月后基本稳定
```

## 使用注意

| 情况 | 处理 |
|------|------|
| 阳虚动风 | 不宜使用（需温阳） |
| 痰热内扰 | 先清热化痰 |
| 脾胃虚寒 | 滋阴药碍胃，需配伍 |
| 孕妇 | 慎用（重镇潜阳） |
