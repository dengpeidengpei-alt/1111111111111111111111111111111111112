# 桑菊饮

## Origin
- **温病条辨** — 吴鞠通
- **分类** — 解表剂·辛凉解表
- **地位** — 风温初起辛凉轻剂，咳嗽主方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|
| 桑叶 | 7.5g | 君药 | 疏散风热，清肺润燥 |
| 菊花 | 3g | 臣药 | 清热解毒，疏风散热 |
| 杏仁 | 7.5g | 臣药 | 宣降肺气，止咳化痰 |
| 连翘 | 5g | 佐药 | 清热解毒，透热转气 |
| 薄荷 | 2.5g | 佐药 | 辛凉透热，疏风散热 |
| 桔梗 | 5g | 佐药 | 宣肺利咽，载药上行 |
| 甘草 | 2.5g | 使药 | 清热解毒，调和诸药 |
| 芦根 | 6g | 佐使 | 清热生津，止渴利尿 |

## 功效
- **疏风清热** — 疏散风热之邪
- **宣肺止咳** — 宣通肺气，止咳化痰

## 方解配伍图

```
桑叶（君）→ 疏散风热，清肺润燥（主入肺经）
     ↕（君臣相须）
菊花（臣）→ 清热解毒，疏风散热（助桑叶散风热）
     ↓
杏仁+桔梗（臣）→ 宣降肺气，止咳化痰（宣肺止咳）
     ↓
薄荷+连翘（佐）→ 辛凉透热，清热解毒（增强解表）
     ↓
芦根（佐使）→ 清热生津，止渴利尿（生津利尿）
     ↓
甘草（使）→ 清热解毒，调和诸药
     ↓
配伍特点：辛凉轻剂，宣肺止咳；风热感冒咳嗽者宜
```

## 主治

### 经典症状（温病条辨）
> 太阴风温，但咳，身不甚热，微渴者，辛凉轻剂桑菊饮主之。

**典型特征**：
- **但咳** — 只咳嗽，不发热或微热（与银翘散区别）
- **身不甚热** — 热度不高，轻度发热
- **微渴** — 略有口渴，热伤津液不重
- **咳嗽为主** — 风温初起以咳嗽为主要症状

### 银翘散与桑菊饮鉴别

| 鉴别点 | 银翘散 | 桑菊饮 |
|--------|--------|--------|
| 主症 | 发热、咽痛 | 但咳、身不甚热 |
| 热势 | 热势较重 | 热势轻微 |
| 病机 | 风热上攻 | 风热犯肺 |
| 治法 | 清热解毒 | 宣肺止咳 |
| 剂量 | 重剂 | 轻剂 |

## 条文精选

### 温病条辨第6条
> 太阴风温，但咳，身不甚热，微渴者，辛凉轻剂桑菊饮主之。

### 温病条辨第7条
> 咳，热伤肺络也。身热甚者，加竹叶、玄参；渴甚者，加天花粉；胸中痞满者，加枳壳、郁金。

## 临床应用（现代医学）

| 疾病 | 机理 | 应用 |
|------|------|------|
| 急性支气管炎 | 宣肺止咳 | 风热咳嗽初起 |
| 感冒 | 疏风清热 | 咳嗽型感冒 |
| 流感 | 辛凉解表 | 轻症流感 |
| 肺炎初期 | 宣肺清热 | 咳嗽发热不甚 |
| 咽炎 | 清热利咽 | 咳嗽咽干 |

## 医案

### 医案1：风热咳嗽
```
患者：女性，28岁
主诉：咳嗽3天，发热38℃
症状：咳嗽痰黄，轻微咽痛，身不甚热
舌象：舌尖红苔薄黄
脉象：脉浮数
辨证：风温犯肺，肺失宣降
治法：疏风清热，宣肺止咳
方药：桑菊饮原方
效果：2剂后咳减，4剂后痊愈
```

### 医案2：流感咳嗽
```
患者：男性，35岁
主诉：流感后咳嗽2周，痰少难咳
症状：咽痒干咳，身微热，口微渴
舌象：舌红苔薄黄
脉象：脉浮数
辨证：风热伤肺，肺阴略伤
治法：疏风清热，润肺止咳
方药：桑菊饮加沙参、麦冬
效果：1周后咳嗽明显减少，继续服用1周痊愈
```

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 桑菊饮 | 原方 | 风温初起辛凉轻剂 |
| 桑菊饮加味 | 加竹叶、玄参 | 热伤肺络，身热甚 |
| 桑菊饮加天花粉 | 加天花粉 | 口渴甚 |
| 桑杏汤 | 去菊花连翘，加栀子、梨皮 | 外感温燥 |
| 银翘散 | 重剂 | 发热咽痛为主 |

## 使用注意

| 情况 | 处理 |
|------|------|
| 发热严重 | 不宜用桑菊饮（热势轻才用） |
| 咳嗽痰多 | 桑菊饮力薄，需加化痰药 |
| 风寒咳嗽 | 不宜使用（需辛温解表） |
| 脾胃虚寒 | 轻剂不宜久用 |
