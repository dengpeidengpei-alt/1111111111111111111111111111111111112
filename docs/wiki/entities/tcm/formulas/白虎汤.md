# 白虎汤

## Origin
- **伤寒论** — 张仲景
- **分类** — 清热剂·清气分热
- **地位** — 阳明气分热盛代表方，大热大渴主方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 生石膏 | 30g | 君药 | 清热泻火，除烦止渴 |
| 知母 | 9g | 臣药 | 清热泻火，滋阴润燥 |
| 甘草 | 3g | 佐药 | 益气和中，调和诸药 |
| 粳米 | 9g | 佐药 | 益胃生津，缓石膏寒性 |

## 功效
- **清热泻火** — 清解气分热邪
- **生津止渴** — 滋阴润燥，止烦渴

## 方解配伍图

```
生石膏（君）→ 清热泻火，除烦止渴（辛寒清气）
     ↕（君臣相须）
知母（臣）→ 清热泻火，滋阴润燥（清热润燥）
     ↓
甘草（佐）→ 益气和中（防石膏寒伤胃气）
     ↓
粳米（佐）→ 益胃生津，缓石膏寒性（护胃生津）
     ↓
配伍特点：辛寒清气，苦寒润燥；清热不伤津，止渴不敛邪
```

## 主治

### 经典症状（伤寒论第219条）
> 三阳合病，腹满身重，难以转侧，口不仁面垢，谵语遗尿。发汗则谵语，下之则额上生汗，手足逆冷。若自汗出者，白虎汤主之。

### 阳明气分热盛四大证
```
大热 — 壮热高热，不恶寒但恶热
大渴 — 大渴引饮，喜冷饮
大汗 — 汗出淋漓，汗量增多
脉洪大 — 脉来洪大有力
```

### 病机分析
```
阳明气分热盛 → 津液损伤 → 大渴大汗
     ↓
热扰心神 → 谵语心烦
     ↓
治法：清热泻火，生津止渴
```

## 条文精选

### 伤寒论第219条

### 伤寒论第26条
> 服桂枝汤，大汗出后，大烦渴不解，脉洪大者，白虎加人参汤主之。

### 温病条辨第7条
> 太阴温病，脉浮洪，舌黄，渴甚，大汗，面赤恶热者，辛凉重剂白虎汤主之。

## 医案

### 医案1：大叶性肺炎
```
患者：男性，42岁
主诉：高热5天，体温40℃，咳嗽胸痛
症状：大渴引饮，汗出淋漓，痰黄带血
舌象：舌红苔黄燥
脉象：脉洪大数
辨证：阳明气分热盛，肺热炽盛
治法：清热泻火，宣肺止咳
方药：白虎汤合麻杏石甘汤
效果：2剂后热退，咳喘减轻，继续调理2周痊愈
```

### 医案2：乙型脑炎
```
患者：儿童，8岁
主诉：高热抽搐2天，体温39.5℃
症状：大汗大渴，面赤烦躁，舌謇弄舌
舌象：舌红苔黄
脉象：脉洪数
辨证：暑温，气分热盛，热极生风
治法：清热泻火，凉肝熄风
方药：白虎汤加羚羊角、钩藤
效果：3剂后热退抽止，继续清热养阴调理
```

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 白虎汤 | 原方 | 阳明气分热盛基础方 |
| 白虎加人参汤 | 加人参 | 热盛伤津，气津两伤 |
| 白虎加桂枝汤 | 加桂枝 | 温疟但热无寒 |
| 白虎加苍术汤 | 加苍术 | 湿温病热重湿轻 |
| 竹叶石膏汤 | 去知母，加竹叶半夏麦冬 | 热病后期余热未尽 |
