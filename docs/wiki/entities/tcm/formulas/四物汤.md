# 四物汤

## Overview
- **出处**: 太平惠民和剂局方
- **分类**: 补益剂 · 补血剂
- **地位**: 妇科补血调经基础方，血证通治之剂

## 组成（经典剂量）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 熟地黄 | 12g | 君药 | 滋阴补血，益精填髓 |
| 当归 | 9g | 臣药 | 补血活血，调经止痛 |
| 白芍 | 9g | 佐药 | 养血柔肝，缓急止痛 |
| 川芎 | 6g | 使药 | 行气活血，通络止痛 |

## 功效
- **补血调血** — 核心功效
- **滋阴润燥** — 熟地黄兼有
- **活血行气** — 川芎当归兼有

## 主治

### 营血虚滞证（经典描述）
- **血虚** — 面色苍白/萎黄，头晕心悸
- **血滞** — 月经不调/痛经，舌淡脉细

### 病机分析
```
营血亏虚 → 脏腑失养 → 功能减退
     ↓
血行不畅 → 瘀血内生 → 疼痛有块
     ↓
治法：补血为主，佐以活血
```

```
│         ↓                    ↓                        │

```

## 类方演变

| 方名 | 变化 | 主治 |
|------|------|------|
| 四物汤 | 原方 | 血虚基础方 |
| 八珍汤 | 四物+四君子 | 气血双虚 |
| 桃红四物汤 | 四物+桃仁、红花 | 血瘀经闭 |
| 胶艾汤 | 四物+阿胶、艾叶 | 崩漏下血 |
| 圣愈汤 | 四物+人参、黄芪 | 气血两虚 |
| 血府逐瘀汤 | 四物+桃红四逆散 | 胸中血瘀 |

## 临床应用

### 妇科
| 疾病 | 应用 | 加减 |
|------|------|------|
| 月经不调 | 基础方 | 经期提前加丹皮、黄芩 |
| 痛经 | 血虚气滞 | 加香附、延胡索 |
| 闭经 | 血瘀 | 加桃仁、红花 |
| 更年期综合征 | 肾阴虚 | 加女贞子、旱莲草 |

### 内科
| 疾病 | 应用 | 加减 |
|------|------|------|
| 贫血 | 血虚 | 加黄芪、人参 |
| 眩晕 | 血虚肝旺 | 加天麻、钩藤 |
| 心悸 | 心血不足 | 加酸枣仁、远志 |

### 外科
| 疾病 | 应用 | 加减 |
|------|------|------|
| 荨麻疹 | 血虚风燥 | 加防风、荆芥 |
| 湿疹 | 血虚湿蕴 | 加苍术、茯苓 |

## 使用注意

| 情况 | 处理 |
|------|------|
| 脾胃虚弱 | 熟地用生地代替，或加陈皮 |
| 湿盛便溏 | 去熟地，加茯苓、薏苡仁 |
| 阴虚火旺 | 熟地改为生地，加丹皮 |
| 孕妇 | 慎用川芎，活血药减量 |
