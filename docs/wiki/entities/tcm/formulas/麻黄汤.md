# 麻黄汤

## Origin
- **伤寒论** — 张仲景
- **分类** — 解表剂·辛温解表
- **地位** — 太阳病发汗峻剂，中病即止

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 麻黄 | 9g | 君药 | 发汗解表，宣肺平喘 |
| 桂枝 | 6g | 臣药 | 解肌发表，助麻黄发汗 |
| 杏仁 | 9g | 佐药 | 宣降肺气，止咳平喘 |
| 甘草 | 3g | 使药 | 调和诸药，缓麻黄之烈 |

## 功效
- **发汗解表** — 峻汗以撤表邪
- **宣肺平喘** — 宣通肺气，平息喘咳

## 方解配伍图

```
麻黄（君）→ 发汗解表，宣肺平喘
   ↕（相须为用）
桂枝（臣）→ 解肌发表，增强发汗
   ↓
杏仁（佐）→ 宣降肺气，止咳平喘
   ↓
甘草（使）→ 调和诸药，缓麻黄烈性
   ↓
配伍特点：发汗峻剂，宣肺平喘；麻桂相须，杏仁佐使
```

## 主治

### 经典症状（伤寒论第3条）
> 太阳病，头痛，发热，身疼，腰痛，骨节疼痛，恶风，无汗而喘者，麻黄汤主之。

**典型特征**：
- **发热** — 或已发热，或未发热
- **恶寒** — 恶风寒更为明显
- **无汗** — 关键鉴别点（桂枝汤有汗）
- **喘** — 肺气不宣致喘
- **身痛** — 寒邪凝滞经脉

### 病机分析
```
病位：太阳经（肌表）
病性：风寒外束，卫阳被遏
治法：发汗解表，宣肺平喘
鉴别：与桂枝汤比较——桂枝汤有汗，麻黄汤无汗
```

## 条文精选

### 伤寒论第1条（太阳病总纲）
> 太阳病，脉浮，头项强痛而恶寒。

### 伤寒论第35条（麻黄汤典型症状）
> 太阳病，头痛，发热，身疼，腰痛，骨节疼痛，恶风，无汗而喘者，麻黄汤主之。

### 伤寒论第36条
> 太阳与阳明合病，喘而胸满者，不可下，宜麻黄汤。

## 临床应用（现代医学）

| 疾病 | 机理 | 应用 |
|------|------|------|
| 感冒 | 发汗解表 | 无汗型风寒感冒 |
| 流感 | 发汗撤邪 | 高热无汗型流感 |
| 肺炎初期 | 宣肺平喘 | 喘咳明显型肺炎 |
| 支气管哮喘 | 宣肺平喘 | 寒邪袭肺型哮喘 |
| 急性肾炎 | 发汗利水 | 风水初起 |

## 医案

### 医案1：流感高热
```
患者：男性，32岁
主诉：发热2天，体温39.2℃，无汗，身痛剧烈
症状：恶寒明显，喘咳，痰白
脉象：浮紧
辨证：太阳伤寒，表实无汗
治法：发汗解表，宣肺平喘
方药：麻黄汤原方
效果：服药后盖被取汗，2小时后汗出热退，次日恢复
```

### 医案2：支气管哮喘
```
患者：女性，45岁
主诉：哮喘发作2小时，喘憋不能平卧
诱因：受寒后发作
症状：痰白清稀，恶寒，无汗
脉象：浮紧
辨证：寒邪袭肺，肺气不宣
治法：辛温解表，宣肺平喘
方药：麻黄汤加苏子、细辛
效果：1剂后喘减，3剂后平稳
```

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 麻黄汤 | 原方 | 太阳伤寒基础方 |
| 麻黄加术汤 | 加苍术 | 寒湿在表 |
| 麻黄杏仁甘草石膏汤 | 去桂枝，加石膏 | 肺热咳喘 |
| 麻黄杏仁薏苡甘草汤 | 去桂枝，加薏苡仁 | 风湿在表 |
| 大青龙汤 | 加石膏、杏仁、姜枣 | 寒热两感 |

## 使用注意

| 情况 | 处理 |
|------|------|
| 表虚自汗 | 禁用（卫阳已虚） |
| 阴虚血少 | 禁用（发汗伤津） |
| 阳明里热 | 禁用（需清下） |
| 中病即止 | 不可久服，防大汗亡阳 |
| 服药禁忌 | 忌生冷、黏滑、肉面 |
