## 关键经验

#### 1. 图片文字问题
**问题**: AI直接生成的图片中文字会乱码
**原因**: AI模型无法精确渲染文字符号，文字区域被当作纹理生成
**解决**: 生成纯视觉图片后，用PIL后处理添加文字

#### 6. 配套代码工具

**design_preview.py** - DESIGN.md预览生成器 v1.1

**design_to_component.py** - DESIGN.md转React组件
```bash
python design_to_component.py Claude-Work/tcm-knowledge/DESIGN.md tcm_components.jsx
```
- 从DESIGN.md提取颜色、字体、组件规范
- 生成styled-components的React组件代码
- 输出包括theme tokens和组件模板

**design_preview.py** - DESIGN.md预览生成器 v1.1
```bash
python design_preview.py tcm           # 生成中医知识库预览
python design_preview.py agent-skills  # 生成技能库预览
python design_preview.py research      # 生成研究库预览
python design_preview.py all           # 生成所有6个预览
python design_preview.py serve         # 启动本地服务器预览
```

功能：
- 解析DESIGN.md的YAML配置
- 生成带颜色/字体/组件的HTML预览页面
- 输出到各知识库目录的design_preview.html
- 支持 `all` 一键生成所有预览
- 支持 `serve` 启动本地服务器预览
