# 半夏泻心汤

## Origin
- **伤寒论** / **金匮要略** — 张仲景
- **分类** — 和解剂·和中消痞
- **地位** — 寒热错杂痞证代表方，脾胃病常用方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 半夏 | 12g | 君药 | 和胃降逆，消痞散结 |
| 干姜 | 9g | 臣药 | 温中散寒，助半夏和胃 |
| 黄芩 | 9g | 臣药 | 清热燥湿，助半夏降逆 |
| 黄连 | 6g | 佐药 | 清热燥湿，泻火解毒 |
| 人参 | 9g | 佐药 | 益气健脾，补虚和中 |
| 大枣 | 6枚 | 佐药 | 补益脾胃，调和营卫 |
| 甘草 | 9g | 使药 | 调和诸药，益气和中 |

## 功效
- **寒热平调** — 核心功效，调和寒热
- **和胃消痞** — 消除心下痞满
- **辛开苦降** — 辛味开散，苦味降泄

## 方解配伍图

```
半夏（君）→ 和胃降逆，消痞散结（降逆和胃）
     ↕（寒热并用）
干姜（臣）→ 温中散寒（温热）
黄芩+黄连（臣）→ 清热燥湿（寒凉）
     ↓
人参+大枣（佐）→ 益气健脾，补虚和中（扶正）
     ↓
甘草（使）→ 调和诸药，益气和中
     ↓
配伍特点：寒热并用，辛开苦降；补泻兼施，标本同治
```

## 主治

### 经典症状（伤寒论第149条）
> 伤寒五六日，呕而发热者，柴胡汤具，但胸胁以下满者，此为痞，大柴胡汤下之则愈，宜半夏泻心汤。

**典型症状**：
- **心下痞满** — 心窝部堵塞胀满，按之不痛
- **呕恶** — 恶心呕吐
- **肠鸣下利** — 肠鸣音亢进，腹泻
- **苔黄腻或白腻** — 寒热错杂之象

### 病机分析
```
表邪内陷 → 中焦痞塞 → 寒热错杂
     ↓
脾胃升降失常 → 呕恶、痞满、肠鸣
     ↓
治法：和中消痞，寒热平调
```

## 条文精选

### 伤寒论第149条
> 伤寒五六日，呕而发热者，柴胡汤具，但以他药下之，柴胡证仍在者，复与柴胡汤。此虽已下之，不为逆，必蒸蒸而振，却发热汗出而解。若心下满而硬痛者，此为结胸，大陷胸汤主之；但满而不痛者，此为痞，柴胡不中与之，宜半夏泻心汤。

### 金匮要略·呕吐哕下利篇
> 呕而肠鸣，心下痞者，半夏泻心汤主之。

## 临床应用（现代医学）

| 疾病 | 机理 | 应用 |
|------|------|------|
| 慢性胃炎 | 寒热平调，和胃降逆 | 幽门螺旋杆菌感染 |
| 胃溃疡 | 和胃消痞，益气健脾 | 寒热错杂型 |
| 功能性消化不良 | 辛开苦降，消痞散结 | 上腹胀满型 |
| 肠易激综合征 | 调和脾胃，止泻 | 寒热错杂型腹泻 |
| 口腔溃疡 | 清热燥湿，温中散寒 | 复发性口疮 |

## 医案

### 医案1：慢性胃炎
```
患者：女性，42岁
主诉：胃脘胀满2年，伴呕恶、肠鸣
症状：心下痞满，按之不痛，苔黄腻
胃镜：慢性浅表性胃炎
辨证：寒热错杂，中焦痞塞
治法：寒热平调，和胃消痞
方药：半夏泻心汤原方
效果：2周后症状减轻，4周后胃镜复查好转
```

### 医案2：口腔溃疡
```
患者：男性，35岁
主诉：口腔溃疡反复发作3年
症状：舌尖溃疡，疼痛难忍，伴胃脘胀满
舌象：舌红苔黄腻
脉象：脉滑数
辨证：上热下寒，中焦痞塞
治法：辛开苦降，清热温中
方药：半夏泻心汤加生石膏、知母
效果：1周后溃疡愈合，继续调理2月未复发
```

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 半夏泻心汤 | 原方 | 寒热错杂痞证基础方 |
| 生姜泻心汤 | 干姜改生姜，加减 | 水热互结，嗳气食臭 |
| 甘草泻心汤 | 加重甘草 | 痞利甚，胃气虚 |
| 附子泻心汤 | 加附子、大黄、黄连 | 寒热错杂兼阳虚 |
| 黄连汤 | 去黄芩，加桂枝 | 上热下寒，腹痛欲吐 |

## 使用注意

| 情况 | 处理 |
|------|------|
| 心下痞满按之痛 | 结胸证，不宜使用 |
| 纯属寒证 | 不宜使用（需温里） |
| 纯属热证 | 不宜使用（需清热） |
| 孕妇 | 慎用（半夏有一定毒性） |
