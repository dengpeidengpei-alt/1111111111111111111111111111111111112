# 银翘散

## Origin
- **温病条辨** — 吴鞠通
- **分类** — 解表剂·辛凉解表
- **地位** — 温病初起代表方，银翘散系主方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 金银花 | 30g | 君药 | 清热解毒，辛凉透表 |
| 连翘 | 30g | 臣药 | 清热解毒，透热转气 |
| 薄荷 | 18g | 佐药 | 辛凉透热，疏风散热 |
| 牛蒡子 | 18g | 佐药 | 疏散风热，利咽解毒 |
| 荆芥穗 | 12g | 佐药 | 辛散风寒，助透表邪 |
| 淡豆豉 | 15g | 佐药 | 解表除烦，宣散风热 |
| 桔梗 | 18g | 佐药 | 宣肺利咽，载药上行 |
| 甘草 | 12g | 使药 | 清热解毒，调和诸药 |
| 竹叶 | 12g | 佐使 | 清热除烦，生津止渴 |
| 芦根 | 30g | 佐使 | 清热生津，止渴利尿 |

## 功效
- **辛凉透表** — 透达温病邪气
- **清热解毒** — 清解温热毒邪

## 方解配伍图

```
金银花（君）→ 清热解毒，辛凉透表
     ↕（君臣相须）
连翘（臣）→ 清热解毒，透热转气
     ↓
薄荷+牛蒡子（佐）→ 疏散风热，利咽
荆芥穗+淡豆豉（佐）→ 辛散风寒，助透表（少量温性反佐）
     ↓
桔梗（佐）→ 宣肺利咽，载药上行
竹叶+芦根（佐使）→ 清热生津，止渴利尿
     ↓
甘草（使）→ 清热解毒，调和诸药
       
配伍特点：清热为主，透表为辅；少量温性反佐，防寒凉遏邪
```

## 主治

### 经典症状（温病条辨）
> 太阴风温、温热、温疫、冬温...但热不恶寒而渴者，辛凉平剂银翘散主之。

**典型特征**：
- **发热** — 但热不恶寒（阳明热盛）
- **口渴** — 口干口渴（热伤津液）
- **咽痛** — 咽喉肿痛（温邪上攻）
- **舌尖红** — 温病舌象（热在卫分）

### 卫气营血辨证定位
```
卫分证（表热）→ 银翘散 ← 温病初起
     ↓
气分证（里热）→ 白虎汤 ← 热盛阳明
     ↓
营分证（热灼）→ 清营汤 ← 热入营分
     ↓
血分证（热瘀）→ 犀角地黄汤 ← 热入血分
```

## 条文精选

### 温病条辨第4条
> 太阴风温、温热、温疫、冬温，初起恶风寒者，桂枝汤主之。但热不恶寒而渴者，辛凉平剂银翘散主之。

### 温病条辨第6条
> 阳明温病，无上焦证，数日不大便，当下之。若其人阴素虚，不可行承气者，增液汤主之。

## 临床应用（现代医学）

| 疾病 | 机理 | 应用 |
|------|------|------|
| 急性上呼吸道感染 | 清热解毒 | 风热感冒初起 |
| 流感 | 辛凉透表 | 发热咽痛型流感 |
| 急性扁桃体炎 | 利咽解毒 | 咽痛明显 |
| 急性支气管炎 | 清热宣肺 | 咳嗽咽干 |
| 麻疹初期 | 辛凉透疹 | 热毒发疹 |

## 医案

### 医案1：风热感冒
```
患者：女性，25岁
主诉：发热2天，体温38.5℃，咽痛口干
症状：但热不恶寒，舌尖红，脉浮数
辨证：温病初起，卫分证
治法：辛凉透表，清热解毒
方药：银翘散原方
效果：2剂后热退，咽痛减轻，继续服用3天痊愈
```

### 医案2：急性扁桃体炎
```
患者：男性，18岁
主诉：咽痛发热2天，体温39℃
症状：扁桃体肿大化脓，口干渴
舌象：舌红苔黄
脉象：脉滑数
辨证：热毒壅结咽喉
治法：清热解毒，利咽消肿
方药：银翘散加马勃、玄参
效果：3剂后扁桃体消肿，一周后痊愈
```

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 银翘散 | 原方 | 风热初起辛凉平剂 |
| 银翘加玄参汤 | 加玄参 | 热伤营阴 |
| 银翘加马勃汤 | 加马勃、牛蒡子 | 咽喉肿痛 |
| 桑菊饮 | 轻剂 | 风温咳嗽（但咳） |
| 麻杏石甘汤 | 重剂 | 肺热咳喘 |

## 使用注意

| 情况 | 处理 |
|------|------|
| 风寒感冒 | 不宜使用（恶寒无汗） |
| 脾胃虚寒 | 慎用（寒凉伤胃） |
| 温病后期 | 需换方（阴伤需滋阴） |
| 服药方法 | 香气大出即取服，勿过煮 |
