# 血府逐瘀汤

## Origin
- **医林改错**（清代·王清任）
- **分类** — 理血剂·活血化瘀
- **地位** — 活血化瘀代表方，胸中血瘀首选方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 桃仁 | 12g | 君药 | 破血祛瘀，润肠通便 |
| 红花 | 9g | 臣药 | 活血化瘀，止痛散结 |
| 当归 | 9g | 臣药 | 补血活血，养血和营 |
| 川芎 | 6g | 佐药 | 活血行气，祛风止痛 |
| 赤芍 | 6g | 佐药 | 清热凉血，活血化瘀 |
| 生地黄 | 9g | 佐药 | 清热凉血，养阴生津 |
| 牛膝 | 9g | 使药 | 活血通经，补肾强腰 |
| 桔梗 | 5g | 使药 | 宣肺利咽，载药上行 |
| 柴胡 | 3g | 使药 | 疏肝解郁，升举清阳 |
| 枳壳 | 6g | 使药 | 行气宽中，消痞除满 |
| 甘草 | 3g | 使药 | 调和诸药，缓急止痛 |

## 功效
- **活血化瘀** — 化解胸中瘀血
- **行气止痛** — 疏肝理气，通络止痛

## 方解配伍图

```
桃仁（君）→ 破血祛瘀，润肠通便（逐瘀要药）
     ↕（相须为用）
红花（臣）→ 活血化瘀，止痛散结（助桃仁活血）
     ↓
当归+川芎（臣）→ 补血活血，行气止痛（和营活血）
赤芍+生地黄（佐）→ 清热凉血，制活血药温燥（凉血养阴）
     ↓
牛膝（使）→ 活血通经，引血下行（通经达下）
桔梗（使）→ 宣肺利咽，载药上行（载药达胸）
     ↓
柴胡+枳壳（使）→ 疏肝解郁，行气宽中（升降气机）
     ↓
甘草（使）→ 调和诸药，缓急止痛
     ↓
配伍特点：活血化瘀为主，行气为辅；升降气机，引血达胸
```

## 主治

### 经典病机（医林改错）
> 胸中血府血瘀之证，所见之证千般，不离血府血瘀。

**典型症状**：
- **头痛** — 血瘀头痛，痛有定处
- **胸痛** — 胸中血瘀，胸痛憋闷
- **心悸** — 血瘀心悸，心神不安
- **失眠** — 血瘀失眠，彻夜不眠
- **呃逆** — 血瘀呃逆，久治不愈

### 病机分析
```
情志内伤 → 气滞血瘀 → 瘀血内停
     ↓
胸中血府 → 心脉痹阻 → 胸痛心悸
     ↓
治法：活血化瘀，行气止痛
```

## 条文精选

### 医林改错·血府逐瘀汤所治症目
> 头痛，胸痛，胸不任物，胸任重物，心悸怔忡，失眠，呃逆，饮水即呛，小儿夜啼...

### 医林改错
> 诸症皆属血瘀，治以活血化瘀，行气止痛。

## 医案

### 医案1：冠心病心绞痛
```
患者：男性，62岁
主诉：胸痛2年，加重1月
症状：胸痛憋闷，心悸气短，舌暗有瘀斑
心电图：ST段改变
辨证：胸中血瘀，心脉痹阻
治法：活血化瘀，行气止痛
方药：血府逐瘀汤加丹参、三七
效果：2周后胸痛减轻，继续服用3月心电图改善
```

### 医案2：血瘀失眠
```
患者：女性，48岁
主诉：失眠5年，彻夜不眠
症状：心烦不安，胸闷刺痛，舌暗有瘀点
脉象：脉涩
辨证：血瘀心脉，心神不安
治法：活血化瘀，安神定志
方药：血府逐瘀汤加琥珀、远志
效果：1月后睡眠改善，继续调理3月基本正常
```

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 血府逐瘀汤 | 原方 | 胸中血瘀基础方 |
| 膈下逐瘀汤 | 加重桃红枳壳 | 腹中瘀血 |
| 通窍活血汤 | 去牛膝桔梗，加麝香葱姜 | 头面血瘀 |
| 少腹逐瘀汤 | 小腹血瘀 | 少腹瘀血痛经 |
| 会厌逐瘀汤 | 咽喉血瘀 | 咽喉肿痛 |

## 使用注意

| 情况 | 处理 |
|------|------|
| 孕妇 | 禁用（活血化瘀） |
| 出血性疾病 | 慎用或加减 |
| 脾胃虚弱 | 可加护胃药 |
| 瘀血后期 | 需配合补益 |
