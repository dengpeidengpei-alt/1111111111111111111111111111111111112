# 桂枝汤

## Origin
- **伤寒论** — 张仲景
- **分类** — 解表剂·辛温解表
- **地位** — 群方之祖，太阳病主方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 桂枝 | 9g | 君药 | 解肌发表，调和营卫 |
| 芍药 | 9g | 臣药 | 敛阴和营，助桂枝调和 |
| 生姜 | 9g | 佐药 | 发散风寒，和胃止呕 |
| 大枣 | 6枚 | 佐药 | 补益脾胃，调和营卫 |
| 甘草 | 6g | 使药 | 调和诸药，益气和中 |

## 功效
- **解肌发表** — 解除肌表之邪
- **调和营卫** — 协调卫阳与营阴

## 方解配伍图

```
桂枝 ←→ 芍药（君臣相使）
  ↓         ↓
生姜 ←→ 大枣（佐使协同）
       ↓
     甘草
       
配伍特点：桂芍相配，调和营卫；姜枣草合用，补益脾胃
```

## 主治

### 经典症状（伤寒论第2条）
> 太阳病，发热，汗出，恶风，脉缓者，名为中风。

- **发热** — 翕翕发热（轻度发热）
- **汗出** — 自汗出（不由自主的出汗）
- **恶风** — 怕风（遇风则不适）
- **脉缓** — 浮而缓（脉浮但不快）

### 病机分析
```
病位：太阳经（肌表）
病性：风邪袭表，卫强营弱
治法：解肌发表，调和营卫
```

## 条文精选

### 伤寒论第1条
> 太阳病，脉浮，头项强痛而恶寒。

### 伤寒论第12条（桂枝汤原方）
> 太阳中风，阳浮而阴弱。阳浮者热自发，阴弱者汗自出。啬啬恶寒，淅淅恶风，翕翕发热，鼻鸣干呕者，桂枝汤主之。

### 伤寒论第53条
> 病常自汗者，此为荣气和。荣气和者，外不谐，以卫气不共荣气谐和故尔。以荣行脉中，卫行脉外，复发其汗，荣卫和则愈，宜桂枝汤。

## 临床应用（现代医学）

| 疾病 | 机理 | 应用 |
|------|------|------|
| 感冒 | 调和营卫 | 汗出倾向的感冒 |
| 流感 | 解肌发表 | 发热汗出型流感 |
| 过敏性鼻炎 | 调和卫气 | 遇风加重的鼻炎 |
| 荨麻疹 | 调和营卫 | 风邪袭表型荨麻疹 |
| 自汗症 | 卫阳不固 | 无热但多汗 |

## 医案

### 医案1：感冒自汗
```
患者：女性，28岁
主诉：感冒3天，发热38℃，汗出不止，恶风
脉象：浮缓
辨证：太阳中风，营卫不和
治法：解肌发表，调和营卫
方药：桂枝汤原方
效果：服1剂后汗止，3剂痊愈
```

### 医案2：过敏性鼻炎
```
患者：男性，35岁
主诉：遇风即打喷嚏，流清涕，晨起明显
病史：反复发作2年
辨证：卫阳不固，荣卫失调
治法：调和营卫，固护卫阳
方药：桂枝汤加黄芪、防风
效果：2周后发作减少，继续服用1月巩固
```

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 桂枝汤 | 原方 | 太阳中风基础方 |
| 桂枝加葛根汤 | 加葛根 | 太阳中风兼项背强 |
| 桂枝加厚朴杏子汤 | 加厚朴、杏仁 | 太阳中风兼喘 |
| 桂枝加桂汤 | 加重桂枝 | 气从心下冲逆 |
| 桂枝加芍药汤 | 加重芍药 | 太阳病误下腹满时痛 |

## 使用注意

| 情况 | 处理 |
|------|------|
| 表实无汗 | 不宜使用（需麻黄汤） |
| 温病初起 | 不宜使用（需银翘散） |
| 服药后啜粥 | 需啜热粥以助药力 |
| 禁生冷黏滑 | 忌口保护脾胃 |
