# 肾气丸

## Origin
- **金匮要略** — 张仲景
- **分类** — 补益剂·补阳剂
- **地位** — 补肾阳代表方，阴中求阳经典

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 干地黄 | 24g | 君药 | 滋阴补肾，益精填髓 |
| 山茱萸 | 12g | 臣药 | 补肝肾，涩精气 |
| 山药 | 12g | 臣药 | 补脾肾，益气阴 |
| 泽泻 | 9g | 佐药 | 利湿浊，泄肾火 |
| 茯苓 | 9g | 佐药 | 利水渗湿，健脾安神 |
| 牡丹皮 | 9g | 佐药 | 清热凉血，活血化瘀 |
| 桂枝 | 3g | 佐药 | 温通经脉，助阳化气 |
| 附子 | 3g | 佐药 | 补火助阳，散寒止痛 |

## 功效
- **温补肾阳** — 核心功效，补益肾气
- **化气行水** — 附子桂枝温阳利水
- **阴中求阳** — 少量温阳药于大量滋阴药中

## 方解配伍图

```
干地黄（君）→ 滋阴补肾，益精填髓（补肾阴基础）
     ↓
山茱萸+山药（臣）→ 补肝脾肾，涩精气（肝脾肾三阴并补）
     ↓
泽泻+茯苓（佐）→ 利湿浊，健脾运（水湿代谢）
牡丹皮（佐）→ 清热凉血，制温燥（防温药过燥）
     ↓
桂枝+附子（佐）→ 少量温阳，化肾气（少火生气）
     ↓
配伍特点：补阴为主，温阳为辅；补而不腻，温而不燥
```

## 主治

### 经典症状（金匮要略）
> 男子消渴，小便反多，以饮一斗，小便一斗，肾气丸主之。

**典型症状**：
- **腰痛** — 腰为肾之府，肾虚则痛
- **小便不利或频数** — 肾气不化
- **消渴** — 饮一溲一（上消下渴）
- **畏寒肢冷** — 肾阳不足
- **舌淡胖苔白** — 阳虚之象

### 病机分析
```
肾阳不足 → 气化无权 → 开关失司
     ↓
水液代谢失常 → 小便不利/消渴
     ↓
治法：温补肾阳，少火生气
```

## 临床应用（现代医学）

| 疾病 | 机理 | 应用 |
|------|------|------|
| 阳痿 | 温补肾阳 | 肾阳虚型性功能障碍 |
| 腰痛 | 补肾壮腰 | 慢性肾炎腰痛 |
| 前列腺增生 | 化气行水 | 肾气不化型癃闭 |
| 尿崩症 | 补肾固涩 | 肾气不固型多尿 |
| 糖尿病 | 温阳化气 | 阴阳两虚型消渴 |

## 医案

### 医案1：阳痿不育
```
患者：男性，38岁
主诉：阳痿2年，婚后未育
症状：畏寒肢冷，腰膝酸软，精神不振
舌象：舌淡苔白
脉象：脉沉细
辨证：肾阳不足，命门火衰
治法：温补肾阳，益精起痿
方药：肾气丸加淫羊藿、枸杞子
效果：3个月后性功能改善，继续调理半年后妻子怀孕
```

### 医案2：尿频遗尿
```
患者：女性，55岁
主诉：尿频3年，夜间尤甚，遗尿
症状：畏寒肢冷，小便清长，腰膝酸软
舌象：舌淡苔白
脉象：脉沉弱
辨证：肾气不固，膀胱失约
治法：温补肾气，固涩止遗
方药：肾气丸加益智仁、桑螵蛸
效果：2周后尿频减少，1月后夜间只需起夜1次
```

## 类方演变

| 方名 | 组成变化 | 主治差异 |
|------|----------|----------|
| 肾气丸（金匮） | 原方 | 补肾阳基础方 |
| 济生肾气丸 | 加牛膝、车前子 | 肾虚水肿 |
| 右归丸 | 去泽泻茯苓，加杜仲菟丝子 | 纯补肾阳 |
| 左归丸 | 去桂枝附子，加枸杞龟板 | 纯补肾阴 |
| 六味地黄丸 | 去桂枝附子 | 肾阴虚基础方 |

## 使用注意

| 情况 | 处理 |
|------|------|
| 阴虚火旺 | 不宜使用（助火伤阴） |

| 外感发热 | 暂停服用 |
