# 补中益气汤

## Origin
- **脾胃论**（金代·李东垣）
- **分类** — 补益剂·补气剂
- **地位** — 补中益气代表方，升阳举陷经典方

## 组成（剂量比例）

| 药材 | 剂量 | 角色 | 功效 |
|------|------|------|------|
| 黄芪 | 18g | 君药 | 补气升阳，益卫固表 |
| 人参 | 9g | 臣药 | 大补元气，健脾益肺 |
| 白术 | 9g | 臣药 | 健脾益气，燥湿利水 |
| 炙甘草 | 9g | 佐药 | 益气补中，调和诸药 |
| 当归 | 6g | 佐药 | 补血活血，养血和营 |
| 陈皮 | 6g | 佐药 | 理气健脾，燥湿化痰 |
| 升麻 | 3g | 使药 | 升阳举陷，引清气上行 |
| 柴胡 | 3g | 使药 | 升提中气，助升麻升阳 |

## 功效
- **补中益气** — 补益中焦脾胃之气
- **升阳举陷** — 升提下陷之中气

## 方解配伍图

```
黄芪（君）→ 补气升阳，益卫固表（补脾益肺）
     ↕（相须为用）
人参+白术（臣）→ 大补元气，健脾益气（助黄芪补气）
     ↓
炙甘草（佐）→ 益气补中，调和诸药（缓诸药）
     ↓
当归+陈皮（佐）→ 补血活血，理气健脾（防补气滞气）
     ↓
升麻+柴胡（使）→ 升阳举陷，引清上行（升提下陷）
     ↓
配伍特点：补气为主，升提为辅；补而不滞，升而不散
```

## 主治

### 经典病机（脾胃论）
> 脾胃气虚，元气不足，则阴火独旺；阴火上冲，则气高而喘，身热而烦，其脉洪大而头痛，或渴不止。

**典型症状**：
- **发热** — 气虚发热（发热汗多，劳累加重）
- **乏力** — 脾胃气虚（倦怠懒言）
- **脱肛** — 中气下陷（气虚失摄）
- **子宫脱垂** — 中气下陷（气虚失固）
- **久泻** — 脾气不升（清阳下陷）

### 病机分析
```
脾胃气虚 → 元气不足 → 阴火内生
     ↓
中气下陷 → 脏腑失摄 → 脱垂崩漏
     ↓
治法：补中益气，升阳举陷
```

## 条文精选

### 脾胃论·饮食劳倦论
> 甘温能除大热，大忌苦寒之药泻胃土耳。今立补中益气汤。

### 脾胃论
> 内伤脾胃，乃伤其气；伤其脾，为热中。补中益气汤主之。

## 医案

### 医案1：脱肛
```
患者：女性，58岁
主诉：脱肛2年，加重1月
症状：排便时直肠脱出，需手送回，倦怠乏力
舌象：舌淡苔白
脉象：脉虚弱
辨证：中气下陷，脾胃气虚
治法：补中益气，升阳举陷
方药：补中益气汤加金樱子、诃子
效果：1月后脱肛发作减少，继续调理3月痊愈
```

### 医案2：气虚发热
```
患者：女性，45岁
主诉：低热3月，体温37.5-38℃
症状：发热汗多，倦怠乏力，劳累后加重
舌象：舌淡苔白
脉象：脉虚弱
辨证：脾胃气虚，阴火内生
治法：补中益气，甘温除热
方药：补中益气汤原方
效果：2周后体温正常，继续服用1月巩固
```
