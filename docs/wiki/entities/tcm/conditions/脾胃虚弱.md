# 脾胃虚弱

## 概述
- **中医病机**: 脾胃运化功能减退
- **主要症状**: 食欲不振、腹胀便溏、乏力倦怠
- **舌脉**: 舌淡苔白，脉虚弱

## 病因病机

```
饮食不节 → 损伤脾胃
    ↓
脾失健运 → 消化不良
    ↓
气血生化不足 → 乏力面色无华
    ↓
脾虚湿盛 → 便溏腹胀
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 脾气虚 | 乏力+食欲不振 | 舌淡苔白，脉弱 | 四君子汤 |
| 脾阳虚 | 畏寒+腹冷痛+喜温 | 舌淡苔白，脉沉迟 | 理中汤 |
| 脾虚湿盛 | 腹胀+便溏+苔白腻 | 舌淡苔白腻，脉濡缓 | 参苓白术散 |
| 中气下陷 | 脱肛+胃下垂+久泻 | 舌淡苔白，脉弱 | 补中益气汤 |

## 经典方剂

### 四君子汤
```python
组成：人参、白术、茯苓、甘草
功效：益气健脾
主治：脾胃气虚基础方
```

### 参苓白术散
```python
组成：人参、白术、茯苓、甘草、山药、莲子、薏苡仁、砂仁、桔梗
功效：健脾渗湿
主治：脾虚湿盛证
```

### 补中益气汤
```python
组成：黄芪、人参、白术、甘草、当归、陈皮、升麻、柴胡
功效：补中益气，升阳举陷
主治：中气下陷证
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 慢性胃炎 | 脾胃气虚 | 加陈皮、半夏 |
| 腹泻 | 脾虚湿盛 | 加薏苡仁、车前子 |
| 便秘 | 脾虚气滞 | 加木香、枳实 |
| 贫血 | 脾胃虚弱 | 加当归、熟地 |

## 与其他证型鉴别

| 证型 | 相同点 | 不同点 |
|------|--------|--------|
| 脾胃虚弱 | 消化不良 | 乏力+腹胀 |
| 脾肾阳虚 | 腹胀+便溏 | 畏寒+腰膝冷 |
| 肝郁犯脾 | 腹胀 | 胁痛+情绪相关 |
