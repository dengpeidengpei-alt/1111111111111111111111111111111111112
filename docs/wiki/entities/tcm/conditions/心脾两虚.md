# 心脾两虚

## 概述
- **中医病机**: 心神失养，脾运不健
- **主要症状**: 心悸失眠、食少便溏、乏力倦怠
- **舌脉**: 舌淡苔白，脉细弱

## 病因病机

```
思虑过度 → 耗伤心神
    ↓
心脾两虚 → 心神失养
    ↓
气血不足 → 心悸失眠
    ↓
脾失健运 → 食少便溏
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 心脾气虚 | 心悸+食少 | 舌淡苔白，脉弱 | 归脾汤 |
| 心脾血虚 | 失眠+头晕 | 舌淡苔白，脉细 | 归脾汤合四物汤 |
| 心脾两虚 | 心悸失眠+食少便溏 | 舌淡苔白，脉细弱 | 归脾汤 |

## 代表方剂

### 归脾汤
```python
组成：黄芪、人参、白术、茯苓、酸枣仁、龙眼肉、木香、当归、远志、甘草、生姜、大枣
功效：益气补血，健脾养心
主治：心脾两虚证
```

### 四君子汤
```python
组成：人参、白术、茯苓、甘草
功效：益气健脾
主治：脾胃气虚基础方
```

### 当归补血汤
```python
组成：黄芪、当归
功效：补气生血
主治：血虚发热证
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 神经衰弱 | 心脾两虚 | 加酸枣仁、远志 |
| 失眠症 | 心神失养 | 加夜交藤、合欢皮 |
| 慢性胃炎 | 脾胃虚弱 | 加陈皮、半夏 |
| 贫血 | 气血不足 | 加熟地、白芍 |
| 月经不调 | 心脾两虚 | 加益母草、香附 |

## 与其他证型鉴别

| 证型 | 相同点 | 不同点 |
|------|--------|--------|
| 心脾两虚 | 心悸失眠 | 食少便溏+乏力 |
| 心肾不交 | 心悸失眠 | 腰膝酸软+潮热 |
| 气血不足 | 乏力 | 无明显心脾症状 |
