# 肝郁气滞

## 概述
- **中医病机**: 肝失疏泄，气机郁滞
- **主要症状**: 胁肋胀痛、情绪抑郁、暖气叹息
- **舌脉**: 舌淡红苔薄白，脉弦

## 病因病机

```
情志不畅 → 肝气郁结 → 疏泄失常
    ↓
气机阻滞 → 胁肋胀痛
    ↓
横逆犯胃 → 嗳气反酸
    ↓
肝郁化火 → 急躁易怒
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 肝气郁结 | 胁肋胀痛，情绪抑郁 | 脉弦 | 柴胡疏肝散 |
| 肝郁化火 | 急躁易怒，口苦口干 | 舌红苔黄，脉弦数 | 丹栀逍遥散 |
| 肝郁血瘀 | 胁痛如刺，舌紫暗 | 脉涩 | 血府逐瘀汤 |
| 肝郁乘脾 | 胁痛+腹泻 | 脉弦缓 | 逍遥散 |

## 代表方剂

### 柴胡疏肝散
```python
组成：陈皮、柴胡、川芎、香附、枳壳、芍药、甘草
功效：疏肝理气，和胃止痛
主治：肝气郁结之胁痛
```

### 逍遥散
```python
组成：柴胡、当归、白芍、白术、茯苓、甘草、薄荷、生姜
功效：疏肝解郁，健脾养血
主治：肝郁血虚脾弱证
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 慢性胃炎 | 肝胃不和 | 加香附、延胡索 |
| 胆囊炎 | 肝胆湿热 | 加金钱草、茵陈 |
| 乳腺增生 | 肝郁痰凝 | 加夏枯草、贝母 |
| 月经不调 | 肝郁血虚 | 加益母草、香附 |

## 衍生疾病

```
肝郁气滞
    ├── → 肝郁化火 → 急躁易怒
    ├── → 肝郁血瘀 → 胁痛如刺
    ├── → 肝郁乘脾 → 腹泻
    └── → 肝郁犯胃 → 嗳气反酸
```
