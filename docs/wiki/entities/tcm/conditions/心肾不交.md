# 心肾不交

## 概述
- **中医病机**: 心火亢盛，肾水不足，水火不济
- **主要症状**: 心烦失眠、腰膝酸软、潮热盗汗
- **舌脉**: 舌红少苔，脉细数

## 病因病机

```
心火亢盛
    ↓
肾水不足 ← 肾阴虚
    ↓
水火不济
    ↓
心肾不交
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 心肾阴虚火旺 | 心烦失眠+腰膝酸软+潮热 | 舌红少苔，脉细数 | 黄连阿胶汤 |
| 心肾阳虚 | 心悸+腰膝冷痛+畏寒 | 舌淡苔白，脉沉弱 | 桂枝加龙骨牡蛎汤 |
| 心肾不交失眠 | 心烦多梦+失眠健忘 | 舌红，脉细数 | 交泰丸 |

## 经典方剂

### 黄连阿胶汤
```python
组成：黄连、黄芩、芍药、阿胶、鸡子黄
功效：滋阴降火，除烦安神

```

### 交泰丸
```python
组成：黄连、肉桂
功效：交通心肾
主治：心肾不交之失眠
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 失眠 | 心肾不交 | 加酸枣仁、远志 |
| 神经衰弱 | 心肾阴虚 | 加五味子、麦冬 |
| 更年期综合征 | 肾阴虚 | 加女贞子、旱莲草 |
| 早泄 | 心肾不交 | 加金樱子、龙骨 |

## 与其他证型鉴别

| 证型 | 相同点 | 不同点 |
|------|--------|--------|
| 心肾不交 | 失眠 | 心烦+腰膝酸软+潮热 |
| 心脾两虚 | 失眠 | 乏力+食欲不振 |
| 肝火扰心 | 心烦失眠 | 口苦+急躁易怒 |
