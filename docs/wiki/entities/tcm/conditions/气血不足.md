# 气血不足

## Definition
- 气虚 + 血虚
- 脏腑功能减退
- 失养于全身

## Signs
- 面色苍白/萎黄
- 头晕乏力
- 心悸气短
- 失眠多梦
- 舌淡苔白

## Causes
- 久病耗伤
- 饮食不节
- 劳倦过度
- 失血过多

## Adjustments
- 补气养血
- 健脾益胃
- 当归补血汤（黄芪+当归）
