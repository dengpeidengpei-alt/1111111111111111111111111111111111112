# 胆郁痰扰

## 概述
- **中医病机**: 胆气郁结，痰热内扰
- **主要症状**: 胆怯易惊、心烦不眠、眩晕呕恶
- **舌脉**: 舌红苔黄腻，脉弦滑

## 病因病机

```
情志不畅 → 胆气郁结
    ↓
胆郁化火 → 炼液为痰
    ↓
痰热内扰 → 心烦不眠
    ↓
胆气不宁 → 胆怯易惊
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 胆郁痰扰 | 胆怯易惊+心烦 | 舌红苔黄腻，脉弦滑 | 温胆汤 |
| 痰热上扰 | 眩晕呕恶+苔黄腻 | 舌红苔黄腻，脉弦滑 | 温胆汤合天麻钩藤饮 |
| 痰迷心窍 | 神志恍惚+喉中痰鸣 | 舌淡苔白腻，脉滑 | 涤痰汤 |
| 胆郁化火 | 口苦+急躁易怒 | 舌红苔黄，脉弦数 | 丹栀逍遥散 |

## 代表方剂

### 温胆汤
```python
组成：半夏、竹茹、枳实、陈皮、茯苓、甘草、生姜、大枣
功效：理气化痰，清胆和胃
主治：胆郁痰扰证
```

### 涤痰汤
```python
组成：半夏、竹茹、胆南星、枳实、茯苓、石菖蒲、人参、甘草
功效：涤痰开窍
主治：痰迷心窍证
```

### 天麻钩藤饮
```python
组成：天麻、钩藤、石决明、川牛膝、杜仲、益母草、桑寄生、夜交藤、茯神
功效：平肝熄风，清热活血
主治：肝阳上亢，肝风上扰
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 神经官能症 | 胆郁痰扰 | 加远志、酸枣仁 |
| 眩晕症 | 痰热上扰 | 加半夏、白术 |
| 失眠 | 痰热内扰 | 加黄连、竹叶 |
| 抑郁症 | 胆气不宁 | 加柴胡、香附 |
| 癫痫 | 痰迷心窍 | 加胆南星、石菖蒲 |

## 与其他证型鉴别

| 证型 | 相同点 | 不同点 |
|------|--------|--------|
| 胆郁痰扰 | 胆怯易惊 | 心烦+苔黄腻 |
| 心肾不交 | 心烦失眠 | 腰膝酸软+舌红少苔 |
| 肝郁气滞 | 胆气郁结 | 胁肋胀痛+脉弦 |
