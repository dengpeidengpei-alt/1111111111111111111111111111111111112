# 湿热下注

## 概述
- **中医病机**: 湿热邪气下注下焦
- **主要症状**: 下部潮湿、瘙痒、黄带、小便短赤
- **舌脉**: 舌红苔黄腻，脉滑数

## 病因病机

```
外感湿热 → 湿热下注
    ↓
蕴结下焦
    ↓
肝经湿热 → 阴部潮湿瘙痒
    ↓
膀胱湿热 → 小便短赤涩痛
    ↓
大肠湿热 → 腹泻脓血
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 肝经湿热 | 阴部潮湿瘙痒 | 舌红苔黄腻，脉弦数 | 龙胆泻肝汤 |
| 膀胱湿热 | 小便短赤涩痛 | 舌红苔黄腻，脉滑数 | 八正散 |
| 大肠湿热 | 腹泻脓血，里急后重 | 舌红苔黄腻，脉滑数 | 白头翁汤 |
| 湿热下注带下 | 黄带增多 | 舌红苔黄腻，脉滑数 | 止带方 |

## 经典方剂

### 龙胆泻肝汤
```python
组成：龙胆草、黄芩、栀子、泽泻、木通、车前子、当归、生地、柴胡、甘草
功效：清肝胆实火，利肝经湿热
主治：肝经湿热下注
```

### 八正散
```python
组成：车前子、瞿麦、扁蓄、滑石、山栀、甘草、木通、大黄
功效：清热利湿，通淋止痛
主治：膀胱湿热证
```

### 白头翁汤
```python
组成：白头翁、黄柏、黄连、秦皮
功效：清热解毒，凉血止痢
主治：热毒血痢
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 尿道炎 | 膀胱湿热 | 加金银花、连翘 |
| 阴道炎 | 肝经湿热 | 加苦参、百部 |
| 痔疮 | 大肠湿热 | 加槐花、地榆 |
| 前列腺炎 | 湿热下注 | 加黄柏、薏苡仁 |

## 鉴别诊断

| 证型 | 相同点 | 不同点 |
|------|--------|--------|
| 湿热下注 | 带下黄稠 | 阴部潮湿+小便短赤 |
| 脾虚湿盛 | 带下 | 便溏+腹胀（不黄） |
| 寒湿下注 | 带下清稀 | 不痒+小便清长 |
