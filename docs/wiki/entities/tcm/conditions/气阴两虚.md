# 气阴两虚

## Definition
- 气虚 + 阴虚
- 热病后期/慢性病常见

## Signs
- 神疲乏力
- 口干咽燥
- 盗汗自汗
- 舌红少苔
- 脉细弱

## Causes
- 热病伤阴
- 久病耗气
- 劳倦伤气
- 失眠多梦

## Adjustments
- 益气养阴
- 生津止渴
- 清补兼施
