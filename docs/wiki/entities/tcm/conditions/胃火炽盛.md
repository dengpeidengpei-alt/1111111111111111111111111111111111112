# 胃火炽盛

## 概述
- **中医病机**: 胃中火热，灼伤津液
- **主要症状**: 胃脘灼痛、口渴喜冷饮、消谷善饥
- **舌脉**: 舌红苔黄，脉滑数

## 病因病机

```
过食辛辣 → 胃热内生
    ↓
胃火炽盛 → 灼伤津液
    ↓
胃热上炎 → 牙龈肿痛
    ↓
热扰神明 → 多食易饥
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 胃火炽盛 | 胃脘灼痛+口渴 | 舌红苔黄，脉滑数 | 清胃散 |
| 胃热上炎 | 牙龈肿痛+口臭 | 舌红苔黄，脉滑数 | 清胃散合玉女煎 |
| 胃阴虚 | 口干+饥不欲食 | 舌红少苔，脉细数 | 益胃汤 |
| 胃热腑实 | 便秘+腹痛 | 舌红苔黄燥，脉滑数 | 大承气汤 |

## 代表方剂

### 清胃散
```python
组成：黄连、升麻、生地、丹皮、当归
功效：清胃凉血
主治：胃火炽盛证
```

### 玉女煎
```python
组成：石膏、熟地、麦冬、知母、牛膝
功效：清胃热，滋肾阴
主治：胃热阴虚证
```

### 大承气汤
```python
组成：大黄、芒硝、枳实、厚朴
功效：峻下热结
主治：阳明腑实证
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 口腔溃疡 | 胃火炽盛 | 加玄参、淡竹叶 |
| 牙龈炎 | 胃热上炎 | 加金银花、连翘 |
| 胃炎 | 胃火炽盛 | 加蒲公英、白花蛇舌草 |
| 糖尿病 | 胃热消谷 | 加天花粉、葛根 |
| 便秘 | 胃热腑实 | 加大黄、芒硝 |

## 与其他证型鉴别

| 证型 | 相同点 | 不同点 |
|------|--------|--------|
| 胃火炽盛 | 胃脘灼痛 | 口渴+消谷善饥 |
| 脾胃湿热 | 胃脘不适 | 苔黄腻+腹胀 |
| 胃阴虚 | 口干 | 饥不欲食+舌红少苔 |
