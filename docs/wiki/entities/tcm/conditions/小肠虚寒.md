# 小肠虚寒

## 概述
- **中医病机**: 小肠阳气不足，失于温煦
- **主要症状**: 腹部隐痛、喜温喜按、肠鸣泄泻
- **舌脉**: 舌淡苔白，脉迟缓

## 病因病机

```
素体阳虚 → 小肠阳虚
    ↓
小肠虚寒 → 失于温煦
    ↓
寒凝气滞 → 腹部隐痛
    ↓
化物失常 → 肠鸣泄泻
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 小肠虚寒 | 腹部隐痛+喜温 | 舌淡苔白，脉迟缓 | 吴茱萸汤 |
| 小肠气痛 | 腹痛+阴囊坠胀 | 舌淡苔白，脉弦 | 天台乌药散 |
| 脾胃虚寒 | 腹痛+泄泻 | 舌淡苔白，脉弱 | 理中汤 |
| 肾阳虚 | 畏寒+五更泄 | 舌淡苔白，脉沉迟 | 四神丸 |

## 代表方剂

### 吴茱萸汤
```python
组成：吴茱萸、人参、生姜、大枣
功效：温中补虚，降逆止呕
主治：肝胃虚寒，浊阴上逆
```

### 理中汤
```python
组成：人参、干姜、白术、甘草
功效：温中祛寒，补气健脾
主治：脾胃虚寒证
```

### 天台乌药散
```python
组成：乌药、木香、小茴香、青皮、高良姜、槟榔、巴豆、川楝子
功效：行气疏肝，散寒止痛
主治：寒凝气滞之疝痛
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 慢性肠炎 | 小肠虚寒 | 加附子、肉桂 |
| 肠易激综合征 | 脾胃虚寒 | 加白芍、陈皮 |
| 腹胀 | 小肠气滞 | 加木香、枳实 |
| 疝气 | 小肠气痛 | 加橘核、荔枝核 |
| 五更泻 | 肾阳虚 | 加补骨脂、肉豆蔻 |

## 与其他证型鉴别

| 证型 | 相同点 | 不同点 |
|------|--------|--------|
| 小肠虚寒 | 腹痛喜温 | 肠鸣泄泻+脉迟缓 |
| 脾胃虚寒 | 腹痛喜温 | 泄泻+食欲不振 |
| 大肠湿热 | 腹痛 | 痢下脓血+苔黄腻 |
