# 肝阳上亢

## 概述
- **中医病机**: 肝阳偏亢，上扰清窍
- **主要症状**: 头晕耳鸣、头痛眩晕、面红目赤
- **舌脉**: 舌红苔黄，脉弦数

## 病因病机

```
肝郁化火 → 灼伤肝阴
    ↓
肝阳上亢 → 上扰清窍
    ↓
肝火上炎 → 头晕头痛
    ↓
热扰心神 → 烦躁易怒
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 肝阳上亢 | 头晕耳鸣+头痛 | 舌红苔黄，脉弦数 | 天麻钩藤饮 |
| 肝火上炎 | 面红目赤+口苦 | 舌红苔黄，脉弦数 | 龙胆泻肝汤 |
| 肝阳化风 | 眩晕震颤+肢体麻木 | 舌红苔黄，脉弦滑 | 镇肝熄风汤 |
| 阴虚阳亢 | 腰膝酸软+舌红少苔 | 舌红少苔，脉细数 | 杞菊地黄丸 |

## 代表方剂

### 天麻钩藤饮
```python
组成：天麻、钩藤、石决明、川牛膝、杜仲、益母草、桑寄生、夜交藤、茯神
功效：平肝熄风，清热活血
主治：肝阳上亢，肝风上扰
```

### 龙胆泻肝汤
```python
组成：龙胆草、黄芩、栀子、泽泻、木通、车前子、当归、生地、柴胡、甘草
功效：清泻肝胆实火
主治：肝胆实火上炎证
```

### 镇肝熄风汤
```python
组成：怀牛膝、代赭石、生龙骨、生牡蛎、龟板、白芍、玄参、天冬、川楝子、茵陈、麦芽、甘草
功效：镇肝熄风，滋阴潜阳
主治：肝阳上亢，肝风内动
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 高血压 | 肝阳上亢 | 加夏枯草、菊花 |
| 头痛 | 肝火上炎 | 加蔓荆子、白芷 |
| 眩晕 | 肝阳化风 | 加半夏、白术 |
| 中风先兆 | 肝阳化风 | 加丹参、红花 |
| 耳鸣耳聋 | 肝阳上亢 | 加磁石、石菖蒲 |

## 与其他证型鉴别

| 证型 | 相同点 | 不同点 |
|------|--------|--------|
| 肝阳上亢 | 头晕头痛 | 面红目赤+脉弦数 |
| 肝郁气滞 | 头晕 | 胁肋胀痛+脉弦 |
| 气血不足 | 头晕 | 乏力+面色苍白 |
