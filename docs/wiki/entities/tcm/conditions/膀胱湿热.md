# 膀胱湿热

## 概述
- **中医病机**: 湿热蕴结膀胱，气化不利
- **主要症状**: 尿频尿急、尿赤涩痛、小腹拘急
- **舌脉**: 舌红苔黄腻，脉滑数

## 病因病机

```
外感湿热 → 下注膀胱
    ↓
膀胱湿热 → 气化不利
    ↓
湿热灼津 → 尿赤涩痛
    ↓
热扰膀胱 → 尿频尿急
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 膀胱湿热 | 尿频尿急+尿赤涩痛 | 舌红苔黄腻，脉滑数 | 八正散 |
| 热淋 | 小便灼热刺痛 | 舌红苔黄，脉滑数 | 八正散合小蓟饮子 |
| 石淋 | 尿中砂石+腰腹绞痛 | 舌红苔黄，脉弦数 | 石韦散合三金汤 |
| 血淋 | 尿血+小腹胀痛 | 舌红苔黄，脉滑数 | 小蓟饮子 |
| 膏淋 | 尿如膏脂+小便涩痛 | 舌红苔黄腻，脉滑数 | 程氏萆薢分清饮 |

## 代表方剂

### 八正散
```python
组成：车前子、瞿麦、扁蓄、滑石、山栀子、甘草、木通、大黄
功效：清热泻火，利水通淋
主治：湿热淋证
```

### 小蓟饮子
```python
组成：小蓟、蒲黄、藕节、滑石、木通、生地、当归、甘草、淡竹叶
功效：凉血止血，利水通淋
主治：血淋
```

### 石韦散
```python
组成：石韦、冬葵子、瞿麦、滑石、车前子
功效：清热利湿，排石通淋
主治：石淋
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 膀胱炎 | 膀胱湿热 | 加金钱草、海金沙 |
| 尿道炎 | 热淋 | 加白花蛇舌草、瞿麦 |
| 肾结石 | 石淋 | 加鸡内金、金钱草 |
| 前列腺炎 | 湿热下注 | 加黄柏、苍术 |
| 尿路感染 | 膀胱湿热 | 加金银花、连翘 |

## 与其他证型鉴别

| 证型 | 相同点 | 不同点 |
|------|--------|--------|
| 膀胱湿热 | 尿频尿急 | 小便涩痛+苔黄腻 |
| 肾阴虚 | 尿频 | 腰膝酸软+舌红少苔 |
| 脾肾阳虚 | 尿频 | 畏寒肢冷+舌淡苔白 |
