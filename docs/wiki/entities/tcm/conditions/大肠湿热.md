# 大肠湿热

## 概述
- **中医病机**: 湿热蕴结大肠，传导失常
- **主要症状**: 腹痛腹泻、里急后重、痢下赤白
- **舌脉**: 舌红苔黄腻，脉滑数

## 病因病机

```
饮食不洁 → 湿热内蕴
    ↓
大肠湿热 → 传导失常
    ↓
气血瘀滞 → 腹痛里急后重
    ↓
热盛肉腐 → 痢下脓血
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 湿热痢 | 腹痛+里急后重+痢下脓血 | 舌红苔黄腻，脉滑数 | 白头翁汤 |
| 热毒痢 | 高热+痢下鲜紫脓血 | 舌红绛，脉滑数 | 白头翁汤合芍药汤 |
| 寒湿痢 | 痢下白多赤少+苔白腻 | 舌淡苔白腻，脉沉缓 | 不换金正气散 |
| 阴虚痢 | 痢下赤白+舌红少苔 | 舌红少苔，脉细数 | 驻车丸 |

## 代表方剂

### 白头翁汤
```python
组成：白头翁、黄连、黄柏、秦皮
功效：清热解毒，凉血止痢
主治：热毒痢证
```

### 芍药汤
```python
组成：芍药、当归、黄连、槟榔、木香、甘草、大黄、黄芩、官桂
功效：清热燥湿，调气和血
主治：湿热痢疾
```

### 葛根芩连汤
```python
组成：葛根、黄芩、黄连、甘草
功效：解表清里
主治：协热下痢
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 细菌性痢疾 | 湿热痢 | 加马齿苋、白头翁 |
| 溃疡性结肠炎 | 大肠湿热 | 加白及、儿茶 |
| 急性肠炎 | 湿热下注 | 加藿香、佩兰 |
| 痔疮 | 大肠湿热 | 加槐角、地榆 |
| 便秘 | 大肠热结 | 加大黄、芒硝 |

## 与其他证型鉴别

| 证型 | 相同点 | 不同点 |
|------|--------|--------|
| 大肠湿热 | 腹泻 | 里急后重+痢下脓血 |
| 膀胱湿热 | 小便不利 | 尿频尿急尿痛 |
| 脾胃湿热 | 苔黄腻 | 腹胀+无里急后重 |
