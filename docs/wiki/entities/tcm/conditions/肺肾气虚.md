# 肺肾气虚

## 概述
- **中医病机**: 肺气不足，肾气不固
- **主要症状**: 久咳喘促、气短乏力、腰膝酸软
- **舌脉**: 舌淡苔白，脉沉弱

## 病因病机

```
久病耗伤 → 肺气不足
    ↓
肺肾气虚 → 肃降失常
    ↓
肾不纳气 → 久咳喘促
    ↓
气短乏力 → 腰膝酸软
```

## 证型分类

| 证型 | 主症 | 舌脉 | 代表方 |
|------|------|------|--------|
| 肺气虚 | 气短懒言+自汗 | 舌淡苔白，脉弱 | 补肺汤 |
| 肾气虚 | 腰膝酸软+夜尿多 | 舌淡苔白，脉沉弱 | 金匮肾气丸 |
| 肺肾气虚 | 久咳喘促+腰膝酸软 | 舌淡苔白，脉沉弱 | 补肺汤合肾气丸 |
| 肾不纳气 | 喘促日久+呼多吸少 | 舌淡苔白，脉细弱 | 人参胡桃汤 |

## 代表方剂

### 补肺汤
```python
组成：黄芪、人参、五味子、熟地、紫菀、桑白皮
功效：补肺益气，止咳平喘
主治：肺气虚证
```

### 金匮肾气丸
```python
组成：地黄、山茱萸、山药、泽泻、茯苓、丹皮、桂枝、附子
功效：温补肾阳
主治：肾阳虚证
```

### 人参胡桃汤
```python
组成：人参、胡桃、生姜
功效：补肾纳气
主治：肾不纳气证
```

## 临床应用

| 疾病 | 应用 | 加减 |
|------|------|------|
| 慢性支气管炎 | 肺气虚 | 加贝母、杏仁 |
| 哮喘 | 肾不纳气 | 加蛤蚧、五味子 |
| 肺气肿 | 肺肾气虚 | 加黄芪、党参 |
| 慢性肾炎 | 肾气虚 | 加益母草、车前子 |
| 前列腺增生 | 肾气虚 | 加王不留行、穿山甲 |

## 与其他证型鉴别

| 证型 | 相同点 | 不同点 |
|------|--------|--------|
| 肺肾气虚 | 喘促+腰膝酸软 | 兼有肺肾两虚 |
| 肾阴虚 | 腰膝酸软 | 潮热盗汗+舌红少苔 |
| 肾阳虚 | 腰膝酸软 | 畏寒肢冷+舌淡苔白 |
