# 黄连

## 分类
- 上品（神农本草经）
- 清热药

## 性味
- 苦、寒

## 归经
- 心、脾、胃、肝、胆、大肠

## 功效
- 清热燥湿
- 泻火解毒

## 主治
- 湿热痞满
- 心胃火盛
- 高热神昏
- 痈肿疔疮

## Usage
- 半夏泻心汤（湿热中焦）
- 黄连解毒汤（热毒炽盛）
- 左金丸（肝火犯胃）
