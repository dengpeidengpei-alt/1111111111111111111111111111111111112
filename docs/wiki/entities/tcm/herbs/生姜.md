# 生姜

## 分类
- 上品（神农本草经）
- 解表药

## 性味
- 辛、微温

## 归经
- 肺、脾、胃

## 功效
- 解表散寒
- 温中止呕
- 温肺止咳

## 主治
- 风寒感冒
- 胃寒呕吐
- 肺寒咳嗽
- 鱼蟹中毒

## Usage
- 桂枝汤（解表散寒）
- 小半夏汤（温中止呕）
- 生姜红糖水（日常保健）
