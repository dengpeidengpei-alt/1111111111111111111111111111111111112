# 黄芩

## 分类
- 中品（神农本草经）
- 清热药

## 性味
- 苦、寒

## 归经
- 肺、胆、脾、大肠

## 功效
- 清热燥湿
- 泻火解毒
- 止血安胎

## 主治
- 湿温暑湿
- 肺热咳嗽
- 血热出血
- 胎热不安

## Usage
- 半夏泻心汤（清热燥湿）
- 黄芩汤（湿热下利）
- 清气化痰丸（肺热）
