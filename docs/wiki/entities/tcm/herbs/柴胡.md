# 柴胡

## 分类
- 上品（神农本草经）
- 解表药（清热解表）

## 性味
- 辛、苦、微寒

## 归经
- 肝、胆、肺

## 功效
- 疏散退热
- 疏肝解郁
- 升阳举陷

## 主治
- 少阳病寒热往来
- 肝郁气滞
- 气虚下陷

## Usage
- 小柴胡汤（少阳病）
- 逍遥散（肝郁）
- 补中益气汤（升阳）
