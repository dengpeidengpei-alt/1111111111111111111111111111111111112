# 麻黄

## 分类
- 中品（神农本草经）
- 解表药

## 性味
- 辛、微苦、温

## 归经
- 肺、膀胱

## 功效
- 发汗解表
- 宣肺平喘
- 利水消肿

## 主治
- 风寒感冒
- 咳嗽气喘
- 水肿

## Usage
- 麻黄汤（太阳伤寒）
- 麻杏石甘汤（肺热咳喘）
- 小青龙汤（外寒内饮）
