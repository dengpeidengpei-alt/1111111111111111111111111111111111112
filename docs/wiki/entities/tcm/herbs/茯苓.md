# 茯苓

## 分类
- 上品（神农本草经）
- 利水渗湿药

## 性味
- 甘、淡、平

## 归经
- 心、肺、脾、肾

## 功效
- 利水渗湿
- 健脾安神

## 主治
- 水肿、痰饮
- 脾虚泄泻
- 心悸、失眠

## Usage
- 五苓散（利水）
- 四君子汤（健脾）
- 归脾汤（安神）
