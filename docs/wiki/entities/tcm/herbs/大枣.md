# 大枣

## 分类
- 上品（神农本草经）
- 补气药

## 性味
- 甘、温

## 归经
- 脾、胃

## 功效
- 补中益气
- 养血安神
- 缓和药性

## 主治
- 脾虚证
- 血虚证
- 脏躁
- 缓和烈药

## Usage
- 桂枝汤（调和营卫）
- 四君子汤（补中益气）
- 甘麦大枣汤（脏躁）
