# 半夏

## 分类
- 中品（神农本草经）
- 化痰药

## 性味
- 辛、温（有毒）

## 归经
- 脾、胃、肺

## 功效
- 燥湿化痰
- 降逆止呕
- 消痞散结

## 主治
- 寒痰、湿痰
- 呕吐、呃逆
- 胸脘痞满
- 瘿瘤瘰疬

## Usage
- 二陈汤（燥湿化痰）
- 小半夏汤（降逆止呕）
- 半夏厚朴汤（梅核气）
