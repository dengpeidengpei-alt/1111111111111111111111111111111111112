# 陈皮

## 分类
- 上品（神农本草经）
- 理气药

## 性味
- 辛、苦、温

## 归经
- 脾、肺

## 功效
- 理气健脾
- 燥湿化痰

## 主治
- 脾胃气滞证
- 咳嗽痰多
- 胸脘胀满

## Usage
- 二陈汤（化痰）
- 陈皮茶（日常保健）
- 逍遥散（疏肝理气）
