# 地黄

## 分类
- 上品（神农本草经）
- 补血药

## 性味
- 生地黄：甘、寒
- 熟地黄：甘、微温

## 归经
- 肝、肾

## 功效
- 生地黄：清热凉血、养阴生津
- 熟地黄：补血滋阴、益精填髓

## 主治
- 生地黄：热病伤阴、血热出血
- 熟地黄：血虚、肾阴虚

## Usage
- 四物汤（熟地）
- 六味地黄丸（熟地）
- 青蒿鳖甲汤（生地）
