# 川芎

## 分类
- 上品（神农本草经）
- 活血化瘀药

## 性味
- 辛、温

## 归经
- 肝、胆、心包

## 功效
- 活血行气
- 祛风止痛

## 主治
- 血瘀气滞痛证
- 头痛、痹痛
- 月经不调

## Usage
- 川芎茶调散（头痛）
- 血府逐瘀汤（活血）
- 日常保健（茶饮）
