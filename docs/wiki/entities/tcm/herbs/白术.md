# 白术

## 分类
- 上品（神农本草经）
- 补气药

## 性味
- 甘、苦、温

## 归经
- 脾、胃

## 功效
- 健脾益气
- 燥湿利水
- 止汗安胎

## 主治
- 脾虚证
- 水肿
- 痰饮
- 表虚自汗

## Usage
- 四君子汤（健脾）
- 补中益气汤（益气）
- 苓桂术甘汤（痰饮）
