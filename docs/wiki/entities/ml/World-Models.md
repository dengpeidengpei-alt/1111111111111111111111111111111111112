# World Models - 世界模型

## 概述

**世界模型 (World Models)** 是能够学习和预测环境动态的生成神经网络模型，使Agent能在其"想象空间"中学习和规划。

核心思想：让AI Agent拥有一个"世界的心灵模型"，用它来模拟环境、预测未来、做出安全决策。

### 三要素
| 要素 | 说明 |
|------|------|
| **表示 (Representation)** | 学习环境的压缩表征 |
| **预测 (Prediction)** | 想象未来状态 |
| **决策 (Decision)** | 基于预测选择动作 |

### 应用场景
- 机器人控制 (Robot Control)
- 游戏AI (Game AI)
- 自动驾驶 (Autonomous Driving)
- 具身智能 (Embodied AI)

---

## 核心概念

### 1. 世界模型 = 环境表示
世界模型本质是对环境动力学的可学习函数：

```
P(s'|s, a) → 给定状态s和动作a，预测下一状态s'
```

### 2. 想象预测 (Imaginative Prediction)
Agent可以在"梦境"中预演动作序列：
```
当前状态 → 想象动作A → 预测状态S₁ → 想象动作B → 预测状态S₂ → ...
```

这允许：
- **离线规划**：不实际执行也能评估动作序列
- **安全探索**：在模拟中试错，避免真实世界风险
- **效率提升**：减少与真实环境交互次数

### 3. 安全决策 (Safe Decision-Making)
通过世界模型可以预测危险后果，避免有害动作：

```
危险动作检测:
  for action in candidate_actions:
    future = world_model.predict(current_state, action)
    if future.is_dangerous:
      discard(action)
```

### POMDP 框架

**部分可观测马尔可夫决策过程 (Partially Observable Markov Decision Process)**

| 元素 | 含义 | 世界模型中的作用 |
|------|------|-----------------|
| S | 状态空间 | 潜在状态z的先验分布 |
| A | 动作空间 | Agent可以执行的动作 |
| T | 状态转移 P(s'|s,a) | MDN-RNN预测未来状态分布 |
| R | 奖励函数 R(s,a) | 指导策略优化 |
| O | 观察概率 P(o\|s) | VAE将高维观察编码为潜在向量 |
| gamma | 折扣因子 | 长期回报权重 |

## 技术实现

### 状态估计方法

| 方法 | 功能 | 说明 |
|------|------|------|
| **VAE** | 观察编码 | 将高维图像压缩为低维潜在向量z |
| **MDN-RNN** | 时序预测 | 预测未来潜在状态分布 (Mixture Density Network) |
| **Controller** | 动作选择 | 紧凑策略网络，直接输出动作 |

### 训练流程

```
1. VAE训练:
   观察o → Encoder → z → Decoder → o'
   目标: 重建观察，最小化重建误差

2. MDN-RNN训练:
   序列(z₁,a₁,z₂,a₂,...,zₜ) → MDN-RNN
   目标: 预测下一时刻潜在状态分布 P(zₜ₊₁ | zₜ, aₜ)

3. Controller训练 (Evolution策略):
   用CMA-ES进化优化策略参数
   奖励: 在模拟环境中评估动作序列
```

### Dreamer算法
现代世界模型代表算法 (Ha & Schmidhuber 2018 → Planck et al 2019-2021):

```python
# Dreamer伪代码
def dreamer_train(env, world_model, policy):
    # 1. 收集真实交互数据
    real_rollout = env.sample_batch()

    # 2. 学习世界模型
    world_model.fit(real_rollout)  # VAE + RNN

    # 3. 在想象空间中训练策略
    imag_rollout = world_model.imagine(policy)
    policy.fit(imag_rollout)  # 梯度反向传播

    return policy
```

## 与 Agentic-RAG 对比

| 维度 | World-Models | Agentic-RAG |
|------|--------------|-------------|
| **核心重点** | 环境模拟与预测 | 知识检索与推理 |
| **预测能力** | 有 — 因果预测未来状态 | 无 — 被动响应查询 |
| **规划能力** | 长期规划，多步想象 | 多跳检索，复杂问答 |
| **主要应用** | 机器人、游戏、自动驾驶 | 问答系统、知识管理 |
| **数据来源** | 环境交互数据 | 外部知识库/文档 |
| **安全性** | 可预判危险动作 | 无内建安全机制 |
| **典型场景** | "如果执行动作A，机器人会摔倒吗？" | "查找关于X的相关文档" |
| **代表工作** | Dreamer, World Models, MuZero | ReAct+Retriever, Self-RAG |

### 互补关系

```
┌─────────────────────────────────────────────────────────────┐
│                      智能 Agent 系统                          │
├──────────────────────┬────────────────────────────────────────┤
│   World Models       │         Agentic-RAG                   │
│   (内部世界模拟)      │         (外部知识检索)                 │
│                      │                                        │
│  "理解环境如何运作"   │        "知道什么信息可用"               │
│                      │                                        │
│  → 预测结果          │        → 检索事实                       │
│  → 规划动作          │        → 生成回答                       │
└──────────────────────┴────────────────────────────────────────┘
```

现代高级Agent通常同时配备：
- **世界模型**：理解动作后果，安全的长期规划
- **RAG系统**：获取外部知识，事实性问答
