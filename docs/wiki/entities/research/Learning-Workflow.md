# Learning Workflow - 学习工作流

## Core Process
```
查找索引 → 调用工具 → 开始工作 → 总结整理建立索引
```

## Key Principles
1. 先看已有什么，不从零开始
2. 不确定时先查索引
3. 优先复用已有的，不重复造轮子

## Skills Usage
- Skills（已有skill直接用，不重复创建）
- 已有研究成果（在已有基础上工作）
- 工具（rtk、agent等）
