### 1. 论语 (Confucian Analects)

| 属性 | 说明 |
|------|------|
| 作者 | 孔子弟子及再传弟子 |
| 篇数 | 20篇，492章 |
| 核心思想 | 仁、义、礼、智、信 |
| 文体 | 语录体，对话式 |
| 重要注本 | 何晏《集解》、朱熹《集注》、刘宝楠《正义》 |

**代表章节**:
- 学而篇：学而时习之
- 为政篇：为政以德
- 八佾篇：八佾舞于庭

### 2. 庄子 (Zhuangzi)

| 属性 | 说明 |
|------|------|
| 作者 | 庄周（约前369-前286） |
| 篇数 | 内篇7 + 外篇15 + 杂篇14 = 33篇 |
| 核心思想 | 逍遥游、齐物论、无为 |
| 文体 | 寓言、重言、危言 |
| 重要注本 | 郭象《注》、成玄英《疏》、王先谦《集解》 |

**代表篇目**:
- 逍遥游：鹏程万里
- 齐物论：万物齐一
- 养生主：庖丁解牛

### 3. 诗经 (Book of Songs)

| 属性 | 说明 |
|------|------|
| 篇数 | 305篇 |
| 分类 | 风（160）+ 雅（105）+ 颂（40） |
| 年代 | 西周初至春秋中叶（前11-前6世纪） |
| 表现手法 | 赋、比、兴 |
| 重要注本 | 毛亨《传》、郑玄《笺》、朱熹《诗集传》 |

**六义**:
```
风雅颂（内容分类）←→ 赋比兴（表现手法）
```

### 4. 史记 (Records of the Grand Historian)

| 属性 | 说明 |
|------|------|
| 作者 | 司马迁（前145-前86） |
| 篇数 | 12本纪 + 10表 + 8书 + 30世家 + 70列传 = 130篇 |

| 体裁 | 纪传体通史 |
| 史学价值 | 正史之祖 |

本纪（帝王）→ 表（年表）→ 书（制度）→ 世家（诸侯）→ 列传（人物）

## Learning Methods - 学习方法

### 1. 注疏传统

```
原始经典（经）
    ↓
传/注（汉代）
    ↓
笺/解（魏晋）
    ↓
疏/正义（唐代）
    ↓
集注/章句（宋代）
    ↓
整理/新注（现代）
```

**注疏层次**:
| 层级 | 功能 | 示例 |
|------|------|------|
| 传 | 阐述经义 | 左丘明《左传》 |
| 注 | 解释字词 | 郑玄《毛诗笺》 |
| 疏 | 疏解注文 | 孔颖达《五经正义》 |
| 集解 | 汇集众说 | 何晏《论语集解》 |

### 2. 今古文差异

| 差异 | 今文经 | 古文经 |
|------|--------|--------|
| 文字 | 隶书 | 篆书/古文 |
| 发现 | 孔壁所藏 | 民间发掘 |
| 传本 | 欧阳、大小夏侯 | 贾徽、刘歆 |
| 特点 | 重义理 | 重名物训诂 |
| 代表 | 公羊、谷梁 | 左传、毛诗 |

### 3. 知人论世

```
知人 → 作者生平、时代背景
    ↓
论世 → 政治环境、学术风气
    ↓
论世 → 文章主旨、艺术特色
```
